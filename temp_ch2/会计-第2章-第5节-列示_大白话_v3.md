## 这段到底在讲什么？

这段讲的是：资产负债表里"存货"这个项目怎么填数字——要把多个会计科目的余额加起来，再减去一些抵减科目，最后填到报表里。

## 你可能会卡在这里

你可能会问：资产负债表的"存货"项目包括哪些内容？

这里容易晕：计划成本核算和售价核算的企业，怎么调整？

你可能会问："受托代销商品"为什么要加进来又减出去？

## 逐一破解

### 卡点1：存货项目包括哪些科目？

资产负债表的"存货"项目，要把下面这些科目的期末余额加起来：

| 科目 | 大白话解释 |
|------|-----------|
| 材料采购 | 已经付款、还在路上的材料 |
| 原材料 | 仓库里的原材料 |
| 库存商品 | 仓库里能卖的产品 |
| 周转材料 | 包装物、低值易耗品等 |
| 委托加工物资 | 拿出去让别人加工的材料 |
| 生产成本 | 车间里正在做的半成品 |
| 发出商品 | 已经发出、还没确认销售的货 |
| 受托代销商品 | 帮别人卖的货 |

然后减去：
- 受托代销商品款（欠委托方的钱）
- 存货跌价准备（已经提的跌价准备）

大白话：把企业所有的"货"加起来——仓库里的、路上的、车间里的、发出去了的、帮别人卖的。但帮别人卖的货，虽然在你仓库，但不是你的，所以要减去欠人家的钱。已经提了跌价的，也要扣掉。

### 卡点2：计划成本核算和售价核算的调整

有些企业不用实际成本记账，而是用计划成本或售价：

| 核算方式 | 怎么调整 |
|---------|---------|
| 材料按计划成本核算 | 加或减材料成本差异 |
| 库存商品按计划成本核算 | 加或减产品成本差异 |
| 库存商品按售价核算 | 减去商品进销差价 |

举个例子：
- 库存商品按计划成本算，余额1000元
- 材料成本差异科目贷方余额100元（表示实际比计划便宜）
- 报表填列 = 1000 - 100 = 900元

大白话：计划成本或售价记账的企业，账上的数不是真正的成本，要把差异调回去。计划成本比实际高的，要减掉差异；比实际低的，要加上差异。售价核算的，要减去毛利部分。

### 卡点3：受托代销商品为什么加进来又减出去？

受托代销商品是企业帮别人卖的货，所有权不属于企业。

- 受托代销商品（资产类）：记录帮别人卖的货的价值
- 受托代销商品款（负债类）：记录欠委托方的钱

两个科目金额相等，一增一减，对存货净额没有影响。

为什么要这样处理？

因为准则要求企业将受托代销商品纳入存货管理，但在资产负债表上要真实反映企业实际拥有的存货。

大白话：帮别人卖的货，虽然在你仓库里，但不是你的。账上先加进来表示"我这有这批货"，再减出去表示"但这批货是别人的"。一加一减，相当于没算进去，只是让你知道有这么回事。

锚点ID: 会计-第2章-第2节-移动加权平均法

## 这段到底在讲什么？

这段讲的是：移动加权平均法的计算公式和它的优缺点。每次进货后重新算一遍平均单价，用这个单价来算发出存货的成本。

## 你可能会卡在这里

你可能会问：移动加权平均法到底怎么算？公式看不懂。

这里容易晕：每次进货都要重新算，具体步骤是什么？

你可能会问：这个方法有什么优缺点？

## 逐一破解

### 卡点1：移动加权平均法的计算公式

三个公式，一步一步来：

公式1：算新的单位成本

存货单位成本 =（原有库存存货的实际成本 + 本次进货的实际成本）÷（原有库存存货数量 + 本次进货数量）

大白话：新单价 =（原来仓库里货的总钱数 + 新进货的总钱数）÷（原来仓库里货的总件数 + 新进货的总件数）

公式2：算发出存货的成本

本次发出存货的成本 = 本次发出存货数量 × 本次发货前存货的单位成本

大白话：发出的货值多少钱 = 发出多少件 × 当时算出来的平均单价

公式3：算月末库存成本

本月月末库存存货成本 = 月末库存存货的数量 × 本月月末存货单位成本

大白话：月底仓库里剩的货值多少钱 = 剩多少件 × 最后算出来的平均单价

### 卡点2：具体计算步骤举例

举个例子：

- 1月1日，库存100件，单价10元，总成本1000元
- 1月10日，进货200件，单价11元，总成本2200元

 第一步：重新算平均单价

新单价 =（1000 + 2200）÷（100 + 200）= 3200 ÷ 300 = 10.67元/件

- 1月15日，发出250件

 第二步：算发出成本

发出成本 = 250 × 10.67 = 2667.5元

 第三步：算月末库存

月末库存数量 = 300 - 250 = 50件
月末库存成本 = 50 × 10.67 = 533.5元

大白话：每次进货后，把新老货混在一起算个平均价。下次发货就按这个平均价算。再进货，再重新算平均价。

### 卡点3：优缺点

| 优点 | 缺点 |
|------|------|
| 能及时了解存货结存情况 | 每次进货都要计算，工作量大 |
| 平均单位成本比较客观 | 不便于随时对存货进行盘点分析 |
| 发出和结存成本较均衡 | 如果进货频繁，计算负担重 |

大白话：好处是随时能知道存货大概值多少钱，价格比较公道。坏处是每次进货都要算一遍，货进得勤，会计就得算得勤。

锚点ID: 会计-第2章-第5节-披露

## 这段到底在讲什么？

这段讲的是：财务报表附注里，关于存货要披露哪些信息——包括存货账面价值、计价方法、跌价准备详情以及用于担保的存货情况。

## 你可能会卡在这里

你可能会问：附注披露和报表列示有什么区别？

这里容易晕：跌价准备要披露哪些具体内容？

你可能会问：用于担保的存货为什么要单独披露？

## 逐一破解

### 卡点1：报表列示和附注披露的区别

| 项目 | 报表列示 | 附注披露 |
|------|---------|---------|
| 位置 | 资产负债表 | 财务报表附注 |
| 内容 | 存货总金额 | 存货的详细构成和背景信息 |
| 目的 | 反映财务状况 | 帮助理解报表数字背后的原因 |

大白话：资产负债表就一行"存货"数字，附注要把这个数字拆开说明——有哪些货、怎么算的、跌了多少、有没有抵押给别人。

### 卡点2：附注中要披露的存货信息

具体要披露以下内容：

| 披露项目 | 具体内容 |
|---------|---------|
| 各类存货账面价值 | 期初多少、期末多少 |
| 发出存货成本方法 | 用先进先出法、加权平均法还是个别计价法 |
| 可变现净值确定依据 | 按合同价还是市场价 |
| 跌价准备计提方法 | 按单个、类别还是合并计提 |
| 当期计提金额 | 本期新提了多少跌价准备 |
| 当期转回金额 | 本期转回了多少跌价准备 |
| 计提和转回原因 | 为什么提、为什么转 |
| 用于担保的存货 | 有多少存货被抵押或担保了 |

大白话：附注就是把存货的"家底"说清楚——有什么货、怎么算账、跌没跌价、跌了多少、有没有拿去做抵押。

### 卡点3：用于担保的存货为什么要单独披露？

用于担保的存货，所有权虽然还是企业的，但使用权受到限制。

| 影响 | 说明 |
|------|------|
| 流动性风险 | 这批货不能随便卖，企业变现能力受影响 |
| 债权人关注 | 银行等债权人要知道哪些资产已被占用 |
| 投资者判断 | 帮助判断企业真实的资产质量 |

大白话：有些货虽然还在你仓库，但已经抵押给银行了，不能随便卖。投资者和债权人要知道这回事，不然会以为你这些货随时能变现。

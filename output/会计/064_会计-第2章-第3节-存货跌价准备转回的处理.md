##这段到底在讲什么？

这段讲存货跌价准备转回的条件、金额计算和会计处理。包含例题2-2，但例题的设计意图比计算本身更重要。

例题2-2选了W7型机器，分两个资产负债表日（6月30日和12月31日），目的不是让你练减法——而是要说明两个核心考点：
- 考点1：可变现净值回升时，转回金额 = 已计提余额 - 当期应保留余额
- 考点2：可变现净值高于成本时，转回以"把余额冲到零"为限，不能确认增值收益

##你可能会卡在这里

- 例题2-2为什么要分6月30日和12月31日两个时间点
- 6月30日转回75万元的计算逻辑是什么
- 12月31日为什么只能转回25万元而不是55万元
- 跌价准备账户的余额变化怎么跟踪
- 转回对资产负债表和利润表分别有什么影响
- 转回和计提的分录为什么正好相反

##逐一破解

**卡点1：例题2-2的设计意图**

W7型机器账面成本500万元，这个"成本不变"是例题的关键前提。整个例题中成本始终是500万元，变的只是可变现净值。

教材设计两个时间点有明确目的：

- **6月30日**：可变现净值从400回升到475，但仍低于成本500 → 说明"减值因素部分消失"，需要部分转回
- **12月31日**：可变现净值从475继续回升到555，已经超过成本500 → 说明"减值因素完全消失"，需要把剩余准备全部冲回

如果只设一个时间点，只能考一个考点。两个时间点才能把"部分转回"和"全额冲回至零"都考到。

**卡点2：跌价准备余额跟踪（T型账思路）**

用T型账跟踪跌价准备账户的余额变化：

```
        存货跌价准备
   借方          贷方
              2×22.12.31 计提 100万
 2×23.6.30 转回 75万
              余额 25万
 2×23.12.31 转回 25万
              余额 0
```

关键点：这个账户是备抵科目，正常余额在贷方。转回时记借方，减少贷方余额。

**卡点3：6月30日为什么转回75万元**

计算逻辑分三步：

步骤1：确定当期应保留的跌价准备余额
- 成本500万元 - 可变现净值475万元 = 25万元
- 这意味着：当前状况下，只需要保留25万元准备就够了

步骤2：确定已计提余额
- 上期已计提100万元

步骤3：差额就是应转回金额
- 应转回 = 已计提余额100万 - 当期应保留25万 = 75万元
- 验证：转回后余额 = 100 - 75 = 25万元，等于当期应保留金额

分录：
借：存货跌价准备　　　　　　　　　　750,000
　贷：资产减值损失——存货减值损失　　750,000

利润表影响：当期资产减值损失减少75万元，利润增加75万元。

**卡点4：12月31日为什么只能转回25万元**

表面上算出来"500 - 555 = -55万元"，似乎应该再转回55万。但这个数字的物理意义是什么？

- 可变现净值555万元 > 成本500万元
- 这意味着存货不仅不需要减值准备，反而"升值"了55万元
- 但存货准则不允许确认未实现的增值收益
- 所以当期应保留的跌价准备余额 = 0（不是-55万）

已计提余额还剩25万元，应保留余额是0，所以最多只能转回25万元。

分录：
借：存货跌价准备　　　　　　　　　　250,000
　贷：资产减值损失——存货减值损失　　250,000

如果错误地转回55万元：
- 跌价准备账户会出现借方余额30万元
- 借方余额意味着"资产增值收益"，这在存货准则中是不允许的
- 谨慎性原则只允许确认减值损失，不允许确认增值收益

**卡点5：两个时间点的财务影响对比**

三个时间点的关键数据对比：

- 2×22年12月31日：可变现净值400万元 → 应保留准备100万元 → 已计提余额从0增至100万元 → 本期计提100万元 → 当期利润减少100万元
- 2×23年6月30日：可变现净值475万元 → 应保留准备25万元 → 已计提余额100万元 → 本期转回75万元 → 当期利润增加75万元
- 2×23年12月31日：可变现净值555万元 → 应保留准备0 → 已计提余额25万元 → 本期转回25万元 → 当期利润增加25万元

累计影响：三次处理合计计提100万元，转回100万元，净影响为0。这说明如果减值因素完全消失，前期确认的损失可以全部冲回。

**卡点6：资产负债表上的存货价值变化**

三个时间点存货的账面价值：

- 2×22年12月31日：成本500万元 - 跌价准备100万元 = 账面价值400万元
- 2×23年6月30日：成本500万元 - 跌价准备25万元 = 账面价值475万元
- 2×23年12月31日：成本500万元 - 跌价准备0 = 账面价值500万元

注意：12月31日可变现净值555万元高于成本500万元，但存货仍按成本500万元列示（升值不确认）。这是"成本与可变现净值孰低"原则的另一面。

##考试陷阱

**错误认知**：存货跌价准备一经计提就不能转回。
**正确理解**：存货跌价准备在符合条件时可以转回。条件是以前减记存货价值的影响因素已经消失，不是在当期造成存货可变现净值高于成本的其他影响因素。

**错误认知**：可变现净值高于成本时，可以把跌价准备全部转回后再确认增值收益。
**正确理解**：转回金额以将存货跌价准备余额冲减至零为限。即使可变现净值远高于成本，也不能确认增值收益，最多把原来计提的准备全部冲回。

**错误认知**：资产负债表日可以提前或延后确定存货的可变现净值。
**正确理解**：企业确定存货的可变现净值应当以资产负债表日的状况为基础确定，既不能提前确定也不能延后确定，且每一个资产负债表日都应当重新确定。

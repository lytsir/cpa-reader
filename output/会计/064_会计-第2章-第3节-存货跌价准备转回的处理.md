##这段到底在讲什么？

这段讲存货跌价准备转回的条件、金额计算和会计处理。

核心规则：
- 跌价准备可以转回，但条件是"以前减记存货价值的影响因素已经消失"
- 转回金额以原已计提的金额为限，最多把余额冲减到零
- 每一个资产负债表日都要重新确定可变现净值

原文包含例题2-2，展示了跌价准备从计提→部分转回→继续转回至零的全过程。

##你可能会卡在这里

- 什么情况下跌价准备可以转回，什么情况下不能
- 转回金额怎么算，为什么不是简单地把差额冲回来
- 例题2-2中6月30日为什么转回75万元而不是100万元
- 12月31日为什么只转回25万元而不是55万元
- 转回的分录和计提的分录有什么区别

##逐一破解

**卡点1：转回的条件**

跌价准备转回必须满足一个严格条件：以前导致存货减值的因素已经消失。不是随便什么价格上涨都可以转回。

可以转回的情况：之前因为市场供过于求导致价格下跌，计提了跌价准备；后来市场回暖，价格回升，原来的供过于求因素消失了；这时可以转回之前多提的跌价准备。

不能转回的情况：之前因为A原因减值，计提了跌价准备；后来B原因导致价格上涨，但A原因并没有消失；这时不能转回，因为"减记存货价值的影响因素"没有消失。

举例：手机因新款上市旧款降价，计提跌价准备。后来旧款成为复古爆款，价格回升，原来新款冲击的因素消失了，可以转回。

**卡点2：转回金额的计算逻辑**

关键公式：
- 当期应计提的跌价准备 = 成本 - 可变现净值（当成本 > 可变现净值时）
- 当期应转回的跌价准备 = 已计提的跌价准备余额 - 当期应保留的跌价准备余额

注意：转回金额不能超过已计提的跌价准备余额，最多冲到零。

举例：成本100元
- 第1年末：可变现净值70元 → 应计提30元 → 跌价准备余额30元
- 第2年末：可变现净值85元 → 应计提15元 → 转回15元（30-15） → 余额15元
- 第3年末：可变现净值110元 → 成本 < 可变现净值，不需要计提 → 应转回15元（把余额冲到零）

不能转回25元（即把可变现净值高于成本的25元也确认收益），因为谨慎性原则不允许确认未实现的增值收益。

**卡点3：例题2-2中6月30日为什么转回75万元**

核心结论：当期应保留25万，已计提100万，差额75万就是应转回金额。

计算分三步：
- 步骤1：成本500万 - 可变现净值475万 = 25万（当期应保留）
- 步骤2：上期已计提100万
- 步骤3：应转回 = 100万 - 25万 = 75万

转回后余额 = 100 - 75 = 25万元，等于当期应保留金额。

分录：
借：存货跌价准备　　　　　　　　　　750,000
　贷：资产减值损失——存货减值损失　　750,000

利润表影响：当期资产减值损失减少75万元，利润增加75万元。

详细数据跟踪见【例题2-2深度解析】。

**卡点4：12月31日为什么只能转回25万元**

核心结论：可变现净值555万元高于成本500万元，但转回以"把余额冲到零"为限，不能确认增值收益。

关键分析：
- 已计提余额还剩25万元
- 当期应保留的跌价准备余额 = 0（可变现净值高于成本，不需要准备）
- 应转回 = 已计提余额25万 - 当期应保留0 = 25万

分录：
借：存货跌价准备　　　　　　　　　　250,000
　贷：资产减值损失——存货减值损失　　250,000

如果错误地转回55万元：跌价准备账户会出现借方余额30万元，相当于确认30万增值收益，违反谨慎性原则。

**卡点5：转回与计提的分录对比**

计提跌价准备时（减值时）：
借：资产减值损失
　贷：存货跌价准备

转回跌价准备时（价值恢复时）：
借：存货跌价准备
　贷：资产减值损失

两个分录正好相反。计提时确认损失，转回时冲减损失。

##例题2-2深度解析

**设计意图**

例题2-2选了W7型机器，成本59始终是500万元不变，变的只是可变现净值。设计两个时间点有明确目的：
- 6月30日：可变现净值从400回升到475，但仍低于成本500 → 考"减值因素部分消失，需要部分转回"
- 12月31日：可变现净值从475继续回升到555，已超过成本500 → 考"减值因素完全消失，需要全额冲回至零"

两个时间点才能把"部分转回"和"全额冲回至零"都考到。

**跌价准备余额跟踪（T型账）**

```
        存货跌价准备
   借方          贷方
              2×22.12.31 计提 100万
 2×23.6.30 转回 75万
              余额 25万
 2×23.12.31 转回 25万
              余额 0
```

这个账户是备抵科目，正常余额在贷方。转回时记借方，减少贷方余额。

**三个时间点的关键数据对比**

- 2×22年12月31日：可变现净值400万元 → 应保留准备100万元 → 已计提余额从0增至100万元 → 本期计提100万元 → 当期利润减少100万元
- 2×23年6月30日：可变现净值475万元 → 应保留准备25万元 → 已计提余额100万元 → 本期转回75万元 → 当期利润增加75万元
- 2×23年12月31日：可变现净值555万元 → 应保留准备0 → 已计提余额25万元 → 本期转回25万元 → 当期利润增加25万元

累计影响：三次处理合计计提100万元，转回100万元，净影响为0。这说明如果减值因素完全消失，前期确认的损失可以全部冲回。

**资产负债表上的存货价值变化**

- 2×22年12月31日：成本500万元 - 跌价准备100万元 = 账面价值400万元
- 2×23年6月30日：成本500万元 - 跌价准备25万元 = 账面价值475万元
- 2×23年12月31日：成本500万元 - 跌价准备0 = 账面价值500万元

注意：12月31日可变现净值555万元高于成本500万元，但存货仍按成本500万元列示（升值不确认）。这是"成本与可变现净值孰低"原则的另一面。

##考试陷阱

**错误认知**：存货跌价准备一经计提就不能转回。
**正确理解**：存货跌价准备在符合条件时可以转回。条件是以前减记存货价值的影响因素已经消失，不是在当期造成存货可变现净值高于成本的其他影响因素。

**错误认知**：可变现净值高于成本时，可以把跌价准备全部转回后再确认增值收益。
**正确理解**：转回金额以将存货跌价准备余额冲减至零为限。即使可变现净值远高于成本，也不能确认增值收益，最多把原来计提的准备全部冲回。

**错误认知**：资产负债表日可以提前或延后确定存货的可变现净值。
**正确理解**：企业确定存货的可变现净值应当以资产负债表日的状况为基础确定，既不能提前确定也不能延后确定，且每一个资产负债表日都应当重新确定。

##这段到底在讲什么？

这段讲的是企业通过外购方式取得固定资产时，成本怎么确定。核心内容包括：第一，外购成本的构成（购买价款、相关税费、运输费、装卸费、安装费、专业人员服务费，不含可抵扣的增值税进项税额）；第二，不需安装和需要安装两种情形的不同处理；第三，一笔款项购入多项没有单独标价的资产时，按公允价值比例分配总成本；第四，超过正常信用条件的分期付款具有融资性质，成本按各期付款额的现值之和确定。

##你可能会卡在这里

- 外购成本中"相关税费"包不包括增值税，为什么进项税额不能算进去
- 不需安装和需要安装的固定资产，为什么一个直接进固定资产、一个先进在建工程
- 一笔款买多样东西没单独标价，怎么分车西西
- 超过正常信用条件的分期付款，为什么成本不是合同总价款
- 未确认融资费用是什么，它和长期应付款有什么关系

##逐一破解

卡点1：增值税进项税额为什么不能算进成本？

why：原文规定外购固定资产的成本不包括按规定可以抵扣的增值税进项税额。

what：增值税进项税额是企业支付或承担的、准予从销项税额中抵扣的增值税额。它并非资产本身的成本，而是一种可以抵扣的税收优惠。

how：例如购买设备价款500,000元，增值税进项税额65,000元，总支付565,000元。固定资产成本只算500,000元，65,000元进项税额单独记入"应交税费——应交增值税（进项税额）"科目，以后可以抵扣销项税额。

卡点2：不需安装和需要安装的资产，账务处理有什么不同？

why：原文规定，外购固定资产分为不需要安装和需要安装两种情形，达到预定可使用状态的时点不同，账务处理也不同。

what：不需安装的资产，购入后即可发挥作用，购入时即达到预定可使用状态，直接记入固定资产。需要安装的资产，必须等安装调试完毕、达到设计要求或合同规定的标准，才能转入固定资产，安装前先记入在建工程。

how：不需安装的账务处理：按应计入固定资产成本的金额，借记固定资产，贷记银行存款、应付票据等。需要安装的账务处理：购入时先记入在建工程，安装完毕达到预定可使用状态后，再从在建工程转入固定资产。

卡点3：一笔款买多样资产怎么分成本？

why：原文规定，企业可能以一笔款项同时购入多项没有单独标价的资产，如果这些资产均符合固定资产的定义并满足确认条件，则应将各项资产单独确认为固定资产，并按各项固定资产公允价值的比例对总成本进行分配。

what：这种情况下不能按"主观估计"或"平均分配"来确定各项资产的成本，必须按公允价值比例分配。

how：例如以1,000万元购入三台设备，没有单独标价。经评估，设备A公允价值400万元，设备B公允价值300万元，设备C公允价值300万元，总公允价值1,000万元。则各设备成本分配如下：
- 设备A成本 = 1,000 × (400/1,000) = 400万元
- 设备B成本 = 1,000 × (300/1,000) = 300万元
- 设备C成本 = 1,000 × (300/1,000) = 300万元

卡点4：超过正常信用条件的分期付款为什么按现值计算成本？

why：原文规定，企业购买固定资产通常在正常信用条件期限内付款，但也会发生超过正常信用条件购买固定资产的经济业务。如果合同中规定的付款期限比较长，超过了正常信用条件，这种情况下购货合同实质上具有融资性质。

what：具有融资性质的购货合同，固定资产成本不能以各期付款额之和确定，而应以各期付款额的现值之和确定。各期实际支付的价款之和与其现值之间的差额确认为未确认融资费用。

how：现值计算需要选择恰当的折现率，这个折现率反映当前市场货币时间价值和延期付款债务特定风险，实质上是供货企业的必要报酬率。账务处理为：购入时按购买价款的现值借记固定资产或在建工程，按应支付金额贷记长期应付款，按其差额借记未确认融资费用。

##例题3-1深度解析

题目概述：2×22年2月1日，甲公司购入一台需要安装的生产用机器设备，增值税专用发票上注明的设备价款为500,000元，增值税进项税额为65,000元，款项已通过银行支付；安装时领用本公司原材料一批，价值30,000元，购进该批原材料时支付的增值税进项税额为3,900元；支付安装工人工资为4,900元。该设备于2×22年3月20日安装调试完成，达到预定可使用状态。

第一步：支付设备价款和增值税

设备价款500,000元计入在建工程，增值税进项税额65,000元单独核算，总支付565,000元。

```
借：在建工程——××设备              500,000
    应交税费——应交增值税（进项税额）  65,000
    贷：银行存款                        565,000
```

第二步：领用原材料和支付安装工人工资

领用原材料价值30,000元，安装工人工资4,900元，合计34,900元计入在建工程。

```
借：在建工程——××设备              34,900
    贷：原材料                        30,000
        应付职工薪酬                  4,900
```

第三步：设备安装完毕达到预定可使用状态

在建工程总金额 = 500,000 + 34,900 = 534,900元，转入固定资产。

```
借：固定资产——××设备              534,900
    贷：在建工程——××设备            534,900
```

关键结论：该固定资产最终成本为534,900元。

##例题3-2深度解析

题目概述：2×15年1月1日，甲公司与乙公司签订购货合同，采用分期付款方式购入一台需要安装的特大型设备。设备价款共计900万元，在2×15年至2×19年的5年内每半年支付90万元，每年的付款日期分别为当年6月30日和12月31日。2×15年1月1日设备开始安装，2×15年12月31日达到预定可使用状态，发生安装费398,530.60元，已用银行存款付讷。假定适用的6个月折现率为10%。

第一步：计算购买价款的现值

每半年支付90万元，共10期。现值 = 900,000 × (P/A, 10%, 10) = 900,000 × 6.1446 = 5,530,140元。

初始入账分录：

```
借：在建工程——××设备            5,530,140
    未确认融资费用              3,469,860
    贷：长期应付款——乙公司          9,000,000
```

其中，未确认融资费用 = 9,000,000 - 5,530,140 = 3,469,860元。

第二步：确定信用期间未确认融资费用的分摊额

分摊逻辑是采用实际利率法，每期确认的融资费用 = 期初应付本金余额 × 实际利率（10%）。

第一期（2×15年6月30日）：
- 期初应付本金余额 = 5,530,140元
- 确认的融资费用 = 5,530,140 × 10% = 553,014元
- 应付本金减少额 = 900,000 - 553,014 = 346,986元
- 期末应付本金余额 = 5,530,140 - 346,986 = 5,183,154元

第二期（2×15年12月31日）：
- 期初应付本金余额 = 5,183,154元
- 确认的融资费用 = 5,183,154 × 10% = 518,315.40元
- 应付本金减少额 = 900,000 - 518,315.40 = 381,684.60元
- 期末应付本金余额 = 5,183,154 - 381,684.60 = 4,801,469.40元

以此类推，直至最后一期尾数调整。

第三步：安装期间的会计处理

2×15年1月1日至2×15年12月31日为设备安装期间，未确认融资费用的分摊额符合资本化条件，计入在建工程成本。

2×15年6月30日的分录：

```
借：在建工程——××设备              553,014
    贷：未确认融资费用            553,014

借：长期应付款——乙公司          900,000
    贷：银行存款                    900,000
```

2×15年12月31日的分录：

```
借：在建工程——××设备              518,315.40
    贷：未确认融资费用            518,315.40

借：长期应付款——乙公司          900,000
    贷：银行存款                    900,000

借：在建工程——××设备              398,530.60
    贷：银行存款                    398,530.60
```

固定资产成本 = 5,530,140 + 553,014 + 518,315.40 + 398,530.60 = 7,000,000元。

```
借：固定资产——××设备            7,000,000
    贷：在建工程——××设备          7,000,000
```

第四步：达到预定可使用状态后的会计处理

2×16年1月1日以后，设备已达到预定可使用状态，未确认融资费用的分摊额不再符合资本化条件，应计入当期损益。

2×16年6月30日的分录：

```
借：财务费用                    480,146.94
    贷：未确认融资费用            480,146.94

借：长期应付款——乙公司          900,000
    贷：银行存款                    900,000
```

以后期间的账务处理与2×16年6月30日相同，此处略。

##考试陷阱

错误认知：外购固定资产的成本包括增值税进项税额。
正确理解：外购固定资产的成本包括购买价款、相关税费、运输费、装卸费、安装费和专业人员服务费，不包括可以抵扣的增值税进项税额。

错误认知：一笔款购入多项没有单独标价的资产，按数量平均分配成本。
正确理解：应按各项固定资产公允价值的比例对总成本进行分配，而不是简单的平均分配。

错误认知：分期付款购买固定资产，固定资产成本就是各期付款额之和。
正确理解：超过正常信用条件的分期付款具有融资性质，固定资产成本应按各期付款额的现值之和确定，各期实际支付价款之和与现值之间的差额确认为未确认融资费用。

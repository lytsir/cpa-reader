##这段到底在讲什么？

这段讲的是存货的确认条件。原文明确：存货必须在符合定义的前提下，同时满足两个条件才能确认：（1）与该存货有关的经济利益很可能流入企业；（2）该存货的成本能够可靠地计量。此外，原文还特别说明了数据资源的确认问题：企业日常活动中持有、最终目的用于出售的数据资源，符合存货定义和确认条件的，应当确认为存货。

##你可能会卡在这里

- "很可能"是什么概率标准？和"可能"有什么区别？
- 数据资源为什么可以确认为存货？不是无形资产吗？
- 二手车为什么算存货而不是固定资产？
- 确认条件和定义有什么区别？

##逐一破解

卡点1："很可能"是什么概率标准？

"很可能"是指概率大于50%但小于等于95%。这是会计中的标准概率术语，具体划分如下：
- 极可能：概率 > 95%
- 很可能：50% < 概率 ≤ 95%
- 可能：5% < 概率 ≤ 50%
- 极小可能：概率 ≤ 5%

存货确认要求"很可能"，意味着经济利益流入的概率必须超过50%。如果概率只是"可能"（例如20%-50%），则不满足确认条件。

卡点2：数据资源为什么可以确认为存货？

数据资源是近年会计新问题。关键在于符合存货定义：日常活动中持有、最终目的用于出售。如果企业日常活动中收集、加工数据，并且最终目的是出售这些数据产品，那么这些数据资源就符合存货定义。

但如果数据资源是企业自己使用的（如客户数据库用于内部营销分析），则不符合"最终目的用于出售"的要求，不应确认为存货。

卡点3：二手车为什么算存货？

因为汽车销售企业购进二手车的目的是出售，符合存货定义中"日常活动中持有以备出售"的要求。虽然二手车是使用过的，但从会计角度看，它的经济利益是通过出售实现的，成本也能可靠计量，因此应作为存货。

原文举例："汽车销售企业在日常活动中购进并用于销售的二手车，应当作为存货进行会计处理"。

卡点4：确认条件和定义有什么区别？

定义回答"是什么"的问题，确认条件回答"什么时候能记账"的问题。

一件东西首先要符合存货定义，然后才能看是否满足确认条件。如果连定义都不符合（如工程物资），就不需要讨论确认条件了。只有符合定义的东西，才需要进一步判断是否满足确认条件。

##考试陷阱

1. 混淆"很可能"和"可能"的概率标准。"很可能"要求 > 50%，"可能"仅要求 > 5%。
2. 认为所有数据资源都是无形资产。正确理解：日常活动中持有、最终目的用于出售的数据资源应确认为存货。
3. 认为二手车不能作为存货。正确理解：销售企业购进用于销售的二手车符合存货定义。
4. 将确认条件和定义混为一谈。正确理解：定义是前提，确认条件是进一步要求。

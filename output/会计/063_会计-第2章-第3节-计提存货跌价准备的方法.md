##这段到底在讲什么？

这段讲3种计提方法、5种减值迹象、4种净值为零的情形，以及一个关键例题（表2-1）。

但原文出这个例题的目的不是让你练习加减法——而是要说明"按类别计提"和"按单个计提"会产生不同的财务结果。这是考试最爱考的点。

##你可能会卡在这里

- 为什么要分三种计提方法，不能全部按单个计提吗
- 表2-1为什么要设计三组商品，每组想考查什么
- 按单个计提和按类别计提的差额到底在哪里
- 为什么第一组按单个计提要提400元，按类别却不用提
- 两种方法对利润表和资产负债表的影响有什么不同

##逐一破解

**卡点1：三种计提方法的选择逻辑**

企业为什么不能全部按单个计提？因为成本效益原则。

- 按单个计提：最精确，但工作量大。每件商品、每批材料都要单独算。大型机械、汽车厂用这种。
- 按类别计提：简化处理，类别内可以盈余抵消减值。钉子、螺丝、纽扣等数量多单价低的用这种。
- 合并计提：同一地区、同一产品系列、相同用途、难以分开的。比如同一条生产线上的多种配套零部件。

**卡点2：表2-1设计意图解码**

原文表2-1不是随便拉了三组数据，每组都有考点：

**第一组（A+B）——"盈余抵消减值"的典型**
- A商品：成本4,000，可变现净值3,600 → 单独减值400
- B商品：成本3,500，可变现净值4,000 → 单独盈余1,500
- 组合计：成本7,500，可变现净值7,600 → 整体不减值

这组的设计意图是让你看到：按类别计提时，B的盈余可以抵消A的减值，结果整个类别不需要计提。而按单个计提时，A必须计提400元。

**第二组（C+D）——"单个和类别结果相同"的情形**
- C商品：成本10,000，可变现净值9,600 → 减值400
- D商品：成本4,500，可变现净值4,400 → 减值100
- 组合计：成本14,500，可变现净值14,000 → 减值500

这组的设计意图是让你看到：当类别内每个商品都减值时，按单个和按类别的结果一样。

**第三组（E）——"只有一种商品"的情形**
- E商品：成本70,000，可变现净值56,000 → 减值14,000

这组的设计意图是让你看到：类别只有一个成员时，按类别就等于按单个。

**卡点3：两种方法的差异对比**

以第一组为例，对比按单个计提和按类别计提的财务影响：

**按单个计提：**
- A商品：账面价值3,600（成本4,000 - 跌价准备400）
- B商品：账面价值3,500（成本3,500，不减值）
- 小计：账面价值7,100
- 利润表影响：计提跌价准备400元，减少当期利润400

**按类别计提：**
- 整个类别账面价值7,500（成本7,500，不减值）
- 利润表影响：不计提，不减少当期利润

**差异：**
- 资产负债表上：按单个8,600元，按类别9,000元（注意是整体，不是单个）
- 利润表上：按单个确认损失400元，按类别不确认损失

这就是考点：两种方法不仅计提金额不同，对当期利润也有影响。按类别计提会"抹平"减值损失。

**卡点4：为什么第一组不计提是对的**

质疑：A明明减值了400元，为什么按类别不用提？

因为按类别计提时，企业是把这两种商品作为一个整体来管理和计量的。B商品多赚了1,500元，A商品少赚了400元，整体还是赚了1,100元。从类别管理的角度，这个类别没有减值，不需要计提。

但要注意：这不是说A商品的减值被"覆盖"了。A商品的可变现净值确实低于成本，只是在类别计提的方法下，这个减值被B的盈余"抵消"了。如果后续A卖出去了，按类别计提时不会单独结转A的准备，而是按比例结转。

**卡点5：减值迹象和净值为零的本质区别**

5种减值迹象：货还能卖，但卖不了那么多了。要计提跌价准备，但不清零。
- 市场价格持续下跌且无回升希望
- 产品成本大于售价
- 原材料不适应新产品且市场价低于账面成本
- 商品过时或消费者偏好改变导致价格下跌
- 其他减值情形

4种净值为零：货完全没价值了，应直接清零。
- 已霉烂变质
- 已过期且无转让价值
- 不再需要且无使用价值和转让价值
- 其他无价值情形

区别：减值是"还有救"，净值为零是"彻底没救"。

**卡点6：同一项存货部分有合同部分没合同**

原文强调：应当分别确定可变现净值，分别与各自成本比较，分别确定跌价准备，不得相互抵销。

举例：100台机器中80台有合同（合同价1,000元），20台无合同（市场价900元）。
- 80台按合同价算可变现净值，与80台的成本比较
- 20台按市场价算可变现净值，与20台的成本比较
- 分别确定是否计提
- 不能技20台的准备去抵减80台的盈余

##考试陷阱

**错误认知**：按类别计提存货跌价准备时，类别内各存货项目的减值可以相互抵销。
**正确理解**：这正是按类别计提的特点。类别内盈余的存货可以抵消减值的存货，这是允许的。但同一项存货中部分有合同、部分无合同的，计提的跌价准备不得相互抵销。

**错误认知**：表2-1中第一组按单个计提不减值，按类别计提也不减值，结果没差别。
**正确理解**：第一组按单个计提时A商品要计提400元，按类别计提时不计提。两种方法对当期利润表的影响不同。

**错误认知**：存货市场价格下跌就一定要计提跌价准备。
**正确理解**：市场价格持续下跌只是减值迹象之一，还要综合判断是否可变现净值低于成本。下跌是暂时的、未来可能回升的，不一定需要计提。

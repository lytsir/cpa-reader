##这段到底在讲什么？

这段专门讲材料存货（原材料、在产品、委托加工材料等）期末怎么计量。核心规则是：材料存货的期末价值取决于它所生产的产成品，而不是材料自身的售价。

两条判断标准：
- 产成品的可变现净值预计高于成本 → 材料按成本计量
- 产成品的可变现净值低于成本 → 材料按可变现净值计量，差额计提跌价准备

##你可能会卡在这里

- 材料存货为什么不看材料自身的售价
- "产成品的可变现净值高于成本"中的成本指的是什么成本
- 怎么判断材料是否需要计提跌价准备
- 材料跌价准备和产成品跌价准备有什么关系

##逐一破解

**卡点1：材料存货为什么不看自身售价**

因为企业持有材料的目的是加工，不是直接出售。材料的价值体现在它能变成什么产品上。

举例：服装厂买了一批布料，成本50元/米。布料在市场上直接卖40元/米，但做成衣服后能卖200元/件，每件衣服用1米布，加工成本30元。

如果按布料自身售价算：
- 布料市场价40元 < 成本50元 → 要计提跌价准备10元

但按产成品算：
- 衣服售价200元 - 加工成本30元 - 销售费用20元 = 150元（可变现净值）
- 衣服成本 = 50 + 30 = 80元
- 150元 > 80元 → 产成品没有减值 → 材料不需要计提跌价准备

结论：材料虽然自身降价了，但做成产品后仍然赚钱，所以材料不减值。

**卡点2："产成品的可变现净值高于成本"中的成本是什么**

原文明确说：这里的"成本"是指产成品的生产成本。

产成品生产成本 = 材料成本 + 至完工时估计将要发生的成本（人工、制造费用等）

举例：一批原材料成本100元，加工成产成品还需投入50元。产成品的生产成本就是150元。

如果产成品可变现净值160元：
- 160 > 150 → 产成品没有减值 → 材料按100元计量

如果产成品可变现净值130元：
- 130 < 150 → 产成品减值了 → 材料需要计提跌价准备

**卡点3：怎么判断材料是否需要计提跌价准备**

两步判断法：

步骤1：算产成品的可变现净值
- 产成品估计售价 - 销售费用 - 相关税费 = 产成品可变现净值

步骤2：比较产成品可变现净值与产成品成本
- 可变现净值 ≥ 成本 → 材料不减值，按成本计量
- 可变现净值 < 成本 → 材料减值，按可变现净值计量

举例：原材料成本100元，加工成产品还需50元。产品售价300元，销售费用30元，税费20元。

- 产成品可变现净值 = 300 - 30 - 20 = 250元
- 产成品成本 = 100 + 50 = 150元
- 250 > 150 → 产成品没减值 → 材料按100元计量

如果产品售价降到150元：
- 产成品可变现净值 = 150 - 30 - 20 = 100元
- 产成品成本 = 150元
- 100 < 150 → 产成品减值了 → 材料需要计提跌价准备

**卡点4：材料跌价准备怎么算**

当产成品减值时，材料的可变现净值 = 产成品估计售价 - 至完工成本 - 销售费用 - 相关税费

材料应计提的跌价准备 = 材料成本 - 材料可变现净值

继续上面的例子：原材料成本100元，产品售价150元，加工成本50元，销售费用30元，税费20元。

- 材料可变现净值 = 150 - 50 - 30 - 20 = 50元
- 材料成本100元 > 可变现净值50元
- 应计提跌价准备 = 100 - 50 = 50元

借：资产减值损失　　　　　　　　50
　贷：存货跌价准备　　　　　　　　50

材料在资产负债表上按50元列示（100 - 50）。

**卡点5：材料跌价准备和产成品跌价准备的关系**

注意：计提的是"材料"的跌价准备，不是"产成品"的跌价准备。因为材料还在仓库里，产成品还没有生产出来。

但判断的依据是"产成品是否减值"——如果产成品没减值，即使材料自身降价了，也不计提跌价准备；如果产成品减值了，即使材料自身没降价，也要计提跌价准备。

##考试陷阱

**错误认知**：材料存货的期末计量看材料自身的市场价格。
**正确理解**：材料存货的期末计量以所生产产成品的可变现净值与成本的比较为基础。即使材料自身降价了，只要产成品没有减值，材料仍按成本计量。

**错误认知**："产成品的可变现净值高于成本"中的成本仅指材料成本。
**正确理解**：这里的成本是指产成品的生产成本，包括材料成本和至完工时估计将要发生的成本。

**错误认知**：材料跌价准备按材料的估计售价减去销售费用计算。
**正确理解**：材料的可变现净值 = 所生产产成品的估计售价 - 至完工时估计成本 - 销售费用 - 相关税费。不能直接用材料自身的售价计算。

##这段到底在讲什么？

这段讲的是外购无形资产初始计量的成本构成、不包括的项目，以及分期付款购买无形资产的现值计量方法。外购无形资产的成本包括购买价款、相关税费以及直接归属于使该项资产达到预定用途所发生的其他支出，如专业服务费用、测试费用等。但不包括广告费、管理费用、达到预定用途后发生的费用等。如果分期付款具有融资性质，无形资产成本应为购买价款的现值，实际支付与现值的差额作为未确认融资费用在信用期内分摊。

##你可能会卡在这里

- 哪些支出算成本，哪些不算
- 分期付款为什么要按现值计量
- 未确认融资费用怎么分摊
- 例题4-1和4-2的分录怎么理解

##逐一破解

卡点1：哪些支出算成本，哪些不算？

why：教材列举了明确包括和明确排除的项目，需要精准区分。

what：计入成本的有三类：购买价款、相关税费、直接归属于使资产达到预定用途的其他支出（如专业服务费、测试费）。不计入成本的有：广告费、管理费用、间接费用、达到预定用途后发生的费用、非必要经营活动的支出。

how：企业买了一项专利，支付价款300万元、税费1万元、律师服务费5万元，这些都计入成本。但之后为推广新产品花了50万元广告费，这不能计入专利成本，应计入当期销售费用。

卡点2：分期付款为什么要按现值计量？

why：分期付款跨期较长时，货币有时间价值，简单加总会高估资产价值。

what：如果分期付款超过正常信用条件（通常1年以内），实质上具有融资性质。此时无形资产的成本不是各期付款的合计，而是这些未来付款按适当折现率折算到现在的价值。各期实际付款与现值的差额是融资费用，需要在信用期内按实际利率法摊销。

how：买一项商标权总价1,000万元，分5年每年末付200万元，银行同期利率5%，年金现值系数4.3295。现值=200万×4.3295=865.9万元。无形资产按865.9万元入账，差额134.1万元作为未确认融资费用，以后5年内逐步确认为财务费用。

卡点3：未确认融资费用怎么分摊？

why：这是实际利率法的应用，每期利息费用不同。

what：每期确认的融资费用=期初融资余额×实际利率。期初融资余额=长期应付款余额-未确认融资费用余额。随着付款进行，融资余额逐年减少，确认的利息费用也逐年减少。

how：第1年期初融资余额865.9万元，利息=865.9×5%=43.3万元。付款200万元中，43.3万元是利息，156.7万元是还本。第2年期初余额=865.9-156.7=709.2万元，利息=709.2×5%=35.46万元。以此类推，最后一年利息9.52万元。

##例题4-1深度解析

题目：甲公司从乙公司购入专利权，实际支付价款300万元，相关税费1万元，专业服务费用5万元。

成本计算：300+1+5=306万元。

分录：
```
借：无形资产——专利权    3,060,000
    贷：银行存款              3,060,000
```

解析：这个例子展示了最基本的购入方式。注意专业服务费5万元是"直接归属于使资产达到预定用途"的支出，必须计入成本。如果这笔钱是购买后的维护费，就不能计入。

##例题4-2深度解析

题目：甲公司购入商标权总计1,000万元，每年末付款200万元，5年付清，利率5%，年金现值系数4.3295。

计算过程：
- 无形资产现值=200×4.3295=865.9万元
- 未确认融资费用=1,000-865.9=134.1万元

初始分录：
```
借：无形资产——商标权      8,659,000
    未确认融资费用          1,341,000
    贷：长期应付款           10,000,000
```

第1年末付款分录：
```
借：长期应付款              2,000,000
    贷：银行存款              2,000,000

借：财务费用                  433,000
    贷：未确认融资费用          433,000
```

解析：初始入账时无形资产按现值865.9万元确认，不是1,000万元。未确认融资费用134.1万元是总利息，在5年内按实际利率法分摊。第1年利息43.3万元，是因为年初"欠"了865.9万元，按5%计算。每次付款200万元中，一部分还利息，一部分还本金。

##这段到底在讲什么？

这段讲的是五种计量属性之间的关系以及如何在实务中选择和应用。教材强调：历史成本是基础，其他计量属性是对历史成本的补充和完善。不同计量属性之间不是互相排斥的，而是针对不同类型的资产和负债、在不同的经济环境下使用的。选择计量属性的核心原则是：既要保证信息的可靠性，又要提高信息的相关性，在两者之间找到平衡。

##你可能会卡在这里

- 历史成本和其他属性是什么关系？是对立还是互补？
- 为什么有时要用公允价值替代历史成本？
- 计量属性之间可以转换吗？比如历史成本转成公允价值？
- 选择计量属性时最重要的是什么？

##逐一破解

卡点1：历史成本和其他属性是什么关系？

**不是对立关系，是互补关系**。

历史成本是基础，因为：
- 几乎所有资产初始确认时都用历史成本（有发票、有合同，客观可验证）
- 历史成本是"锚"——不管后续怎么变，初始金额是确定的

其他属性是补充，因为：
- 历史成本不能反映当前价值（10年前买的房子现在值1000万，账上还是100万）
- 某些资产没有历史成本（盘盈资产、捐赠资产）
- 某些负债需要反映当前义务（预计负债、养老金义务）

**关系图**：

```
历史成本（基础，初始计量用）
    ├── 后续计量可能转为：
    │   ├── 公允价值（金融资产、投资性房地产）
    │   ├── 可变现净值（存货减值时）
    │   ├── 现值（长期应收款减值时）
    │   └── 重置成本（盘盈资产）
    └── 特殊情况下直接用：
        ├── 现值（分期付款购买、预计负债）
        ├── 公允价值（捐赠资产、非货币性资产交换）
        └── 重置成本（盘盈资产）
```

卡点2：为什么有时要用公允价值替代历史成本？

因为历史成本在特定情况下会严重失真：

**情况1：市场价格大幅波动**
- 企业持有上市公司股票，买的时候10元/股，现在100元/股
- 如果按历史成本计量，账上还是10元，完全不能反映真实价值
- 按公允价值计量，账上记100元，投资者能看到真实价值

**情况2：资产用途改变**
- 企业原来自用的大楼，后来改为出租（变成投资性房地产）
- 自用固定资产按历史成本折旧，投资性房地产可以选择按公允价值计量
- 改为公允价值后，大楼的当前价值能在报表上反映

**情况3：金融工具**
- 交易性金融资产（短期买卖的股票债券）目的是赚取价差
- 如果按历史成本计量，账上一直不变，完全看不到盈亏
- 按公允价值计量，每个期末按市场价格调整，盈亏及时反映

但公允价值不是万能的：
- 没有活跃市场的资产，公允价值很难确定（主观性强）
- 公允价值波动大，可能导致利润大幅波动
- 确定公允价值需要专业评估，成本较高

卡点3：计量属性之间可以转换吗？

**可以，但有严格限制**。

**允许转换的情况**：

1. **资产重分类**
   - 自用固定资产改为投资性房地产，可以从历史成本模式转为公允价值模式
   - 持有至到期投资重分类为可供出售金融资产，从摊余成本转为公允价值

2. **会计准则要求**
   - 新准则实施后，某些资产必须从历史成本改为公允价值计量

3. **资产性质改变**
   - 存货改为自用（如房地产开发企业把商品房改为办公楼），计量属性相应改变

**不允许转换的情况**：

1. **随意变更**
   - 不能因为今年公允价值涨了就用公允价值，明年跌了又改回历史成本
   - 变更必须有正当理由，且要追溯调整或充分披露

2. **选择性使用**
   - 不能对同类资产一部分用历史成本、一部分用公允价值（除非准则允许）

**转换时的处理**：
- 公允价值 > 账面价值：差额计入其他综合收益或留存收益（不确认收益，防止操纵利润）
- 公允价值 < 账面价值：差额计入当期损失（谨慎性要求）

卡点4：选择计量属性时最重要的是什么？

最重要的原则是**可靠性与相关性的平衡**：

- **优先保证可靠性**：如果信息不可靠（无法验证、主观性太强），再相关也没用
- **尽量提高相关性**：在可靠性不受损的前提下，选择更能反映当前价值的属性
- **考虑成本效益**：确定计量属性的成本不能超过信息带来的效益

**具体选择流程**：

```
第一步：看准则规定
    ├── 准则明确规定用某种属性 → 直接用
    └── 准则允许选择 → 进入第二步

第二步：评估可靠性
    ├── 能可靠取得 → 进入第三步
    └── 不能可靠取得 → 用历史成本或简化处理

第三步：评估相关性
    ├── 历史成本已足够相关 → 用历史成本（成本低）
    └── 需要反映当前价值 → 用公允价值/现值等

第四步：考虑成本效益
    ├── 确定成本合理 → 采用
    └── 确定成本过高 → 简化处理
```

**陷阱提示**：考试中常见的陷阱是"公允价值总是优于历史成本"——这是错的。两种属性各有优劣，选择要看具体情况。另一个陷阱是"计量属性可以随意变更"——这也是错的。变更必须有正当理由，且要按准则规定处理。

##考试陷阱

1. 将历史成本和其他属性关系与相关概念混为一谈。正确理解：二者有本质区别，考试中必须区分清楚。
2. 对有时要用公允价值替代历史成本理解不到位，只做表面记忆。正确理解：必须掌握其内在逻辑和实质含义。
3. 在计量属性之间可以转换吗相关选项中只记结论。正确理解：必须结合定义要求和确认条件进行判断。
4. 忽略选择计量属性时最重要的的适用范围和限制条件。正确理解：必须结合具体情境分析，不能绝对化。

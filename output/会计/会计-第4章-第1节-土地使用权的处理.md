##这段到底在讲什么？

这段讲的是企业取得土地使用权的会计处理规则，包括一般情况下的无形资产确认、自行开发建造厂房时的分别核算、房地产开发企业和外购房屋建筑物的特殊处理，以及改变用途转为投资性房地产的情形。土地使用权通常作为无形资产核算，但与地上建筑物的关系需要区分不同情况处理。

##你可能会卡在这里

- 土地使用权和地上建筑物为什么要分开核算
- 房地产开发企业为什么要把土地使用权计入房屋成本
- 外购房屋建筑物中土地和建筑物价值怎么分配
- 例4-4的分录怎么理解

##逐一破解

卡点1：土地使用权和地上建筑物为什么要分开核算？

why：很多人 intuitively 觉得土地和房子是一体的，应该合并在一起。

what：土地使用权和地上建筑物的经济实质不同。土地使用权是有期限的权利（如50年），地上建筑物是有形资产（如25年）。两者的使用寿命不同，折旧/摊销年限也不同。如果合并核算，会导致摊销/折旧期限不匹配，不能真实反映资产的消耗过程。

how：A公司买了一块地，使用年限50年，在上面建厂房，厂房使用年限25年。土地使用权每年摊销=地价÷50，厂房每年折旧=厂房成本÷25。两者分别核算，分别计提，互不干扰。

卡点2：房地产开发企业为什么要把土地使用权计入房屋成本？

why：房地产企业的经营模式和一般企业不同。

what：一般企业取得土地是为了长期使用（自用），所以作为无形资产。房地产开发企业取得土地是为了建造对外出售的房屋，土地是房屋成本的组成部分。如果把土地单独作为无形资产摊销，而房屋作为存货（开发产品），会导致会计处理逻辑混乱。

how：房地产企业买了地花了1亿元，建楼花了3亿元。这1亿元地价直接计入房屋建筑物成本，最终作为开发产品（存货）的成本，出售时转入主营业务成本。不是作为无形资产摊销50年。

卡点3：外购房屋建筑物中土地和建筑物价值怎么分配？

why：买一栋楼时，价款中包含了土地和建筑物两部分价值，需要合理拆分。

what：如果能合理分配（比如按公允价值比例），就把总价款在土地使用权和地上建筑物之间分配，分别确认为无形资产和固定资产。如果确实无法合理分配，就全部作为固定资产处理。

how：企业花5,000万元买了一栋办公楼。评估显示土地使用权公允价值2,000万元，建筑物公允价值3,000万元。则土地使用权=5,000×2,000/(2,000+3,000)=2,000万元，建筑物=3,000万元。如果无法评估分离，就全部按5,000万元作为固定资产。

卡点4：改变土地使用权用途怎么办？

why：企业最初买地可能是自用，后来改为出租，资产的性质发生了变化。

what：企业改变土地使用权用途，将其用于出租或增值目的时，应将其从无形资产转为投资性房地产。这不是简单的科目重分类，而是资产性质的转换，需要按照投资性房地产的准则进行处理。

how：企业原来把一块地作为无形资产核算，账面价值1,000万元。后来决定把这栋楼出租出去，此时土地使用权也应转为投资性房地产，按投资性房地产准则进行后续计量（成本模式或公允价值模式）。

##例4-4深度解析

题目：2×25年1月1日，A公司购入土地使用权，支付8,000万元，使用年限50年。在该土地上自行建造厂房，材料支出12,000万元，工资8,000万元，其他费用10,000万元。厂房完工，使用年限25年，无净残值，直线法摊销和折旧。

关键要点：
- 土地使用权和厂房分别核算
- 土地使用权50年摊销，厂房25年折旧
- 厂房建设期间土地使用权的摊销计入在建工程成本（本题略）

分录解析：

（1）支付土地转让价款：
```
借：无形资产——土地使用权    80,000,000
    贷：银行存款              80,000,000
```

（2）建造厂房（归集成本）：
```
借：在建工程                300,000,000
    贷：工程物资              120,000,000
        应付职工薪酬           80,000,000
        银行存款             100,000,000
```

（3）厂房达到预定可使用状态：
```
借：固定资产                300,000,000
    贷：在建工程              300,000,000
```

（4）每年摊销和折旧：
- 土地年摊销额=8,000万÷50=160万元
- 厂房年折旧额=30,000万÷25=1,200万元

```
借：制造费用（土地摊销）      1,600,000
    制造费用（厂房折旧）     12,000,000
    贷：累计摊销               1,600,000
        累计折旧              12,000,000
```

解析：厂房建设期间的"三通一平"等费用计入在建工程。本题简化了厂房建设期间土地使用权摊销计入在建工程的处理（教材提示"此题略"）。厂房完工后，土地使用权继续按50年摊销，厂房按25年折旧，两者年限不同但都是按直线法计算。

这段到底在讲什么？

企业每年至少得彻底盘点一次固定资产，看看账上记的数和实际在的东西对不对得上，多了少了都要查明原因并按规矩处理。

你可能会问：盘盈和盘亏分别走什么科目？为什么不一样？

这里容易晕：东西多了和少了，记账方向完全相反，容易记混。

盘亏时，先把固定资产的账面价值转入"待处理财产损溢"，借方记累计折旧和减值准备，贷方记固定资产原价。批准后，能收回的赔偿记"其他应收款"，净损失记"营业外支出——盘亏损失"。

盘盈则完全不同，它属于前期差错，不通过"待处理财产损溢"，而是直接按重置成本借记"固定资产"，贷记"以前年度损益调整"。因为盘盈说明以前年度漏记账了，不是当期正常经营造成的。

你可能会问：盘盈为什么要作为前期差错？

这里容易晕：东西多了，不是好事吗？为什么当成错误处理？

固定资产是大件，管理规范的企业不应该莫名其妙多出资产。盘盈通常是因为以前年度购入时漏记账、已报废未销账的资产又找到了、或者资产转移时手续不全等原因造成的。本质上都是以前年度的账务处理有问题，所以按前期差错处理，调整以前年度的损益，而不是计入当期的营业外收入。

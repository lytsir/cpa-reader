这段到底在讲什么？

盘点时发现多了固定资产，按前期差错处理，先按重置成本入账，再把相关所得税和留存收益调整到位。

你可能会问：具体分录怎么做？先后顺序是什么？

这里容易晕：以前年度损益调整这个科目，到底怎么结平？

分录例1：盘盈一台设备，重置成本1000000元。

借：固定资产 1000000
贷：以前年度损益调整 1000000

分录例2：确认应交所得税，税率25%。

借：以前年度损益调整 250000
贷：应交税费——应交所得税 250000

分录例3：将余额转入留存收益，假设按10%提盈余公积。

借：以前年度损益调整 750000
贷：盈余公积 75000
利润分配——未分配利润 675000

你可能会问：为什么盘盈要补交所得税？

这里容易晕：东西多了，怎么还要交钱？

因为盘盈增加了以前年度的利润，而利润是要交企业所得税的。虽然这是以前年度的事，但税务上仍然要补确认应纳税所得额，补提应交所得税。最后剩下的净利润部分，再按企业利润分配政策转入盈余公积和未分配利润。

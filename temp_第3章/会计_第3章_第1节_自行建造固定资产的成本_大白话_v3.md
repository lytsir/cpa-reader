# 自行建造固定资产的成本

## 核心提炼

这段到底在讲什么？一句话：企业自己盖厂房、装设备，从动工到能用的过程中，哪些花费要算进固定资产成本里，自营和出包两种方式分别怎么记账。

## 卡点引导

你可能会问：自己建的东西，成本不就是把花的钱加起来吗？试运行时卖出去的产品收入，能不能冲减建造成本？

这里容易晕：工程还没办完竣工结算，但已经达到可使用状态了，这时候该怎么入账？折旧怎么提？

还有一个坑：出包建工程，待摊支出是什么？怎么分摊到各个单项工程上？安全生产费又该怎么处理？

## 逐一破解

### 卡点一：自己建造固定资产，成本到底包括哪些？

自行建造固定资产的成本，由建造该项资产达到预定可使用状态前所发生的必要支出构成。具体来说包括：工程用物资成本、人工成本、缴纳的相关税费、应予资本化的借款费用，以及应分摊的间接费用。另外，测试固定资产可否正常运转而发生的支出，也属于达到预定可使用状态前的必要支出，要计入固定资产成本。

注意，测试可否正常运转指的是评估设备的技术和物理性能是否达标，比如能不能生产出合格产品、能不能正常对外出租等，不包括评估这台设备以后能赚多少钱这类财务业绩指标。

关于试运行销售：企业在固定资产达到预定可使用状态前，把试运行产出的产品或副产品对外销售的，应当按照收入和存货准则分别处理，把试运行相关的收入和成本计入当期损益，不能把收入抵掉成本后的净额用来冲减固定资产成本。试运行产出的产品在对外销售前，符合存货条件的先确认为存货，符合其他资产确认条件的确认为相关资产。至于试运行销售属于日常活动还是非日常活动，要看具体情况，日常活动的在营业收入和营业成本列示，非日常活动的在资产处置收益等项目列示。

### 卡点二：自营建造和出包建造有什么区别？

自营建造就是企业自己买材料、自己找人施工，成本按直接材料、直接人工、直接机械施工费来计量。工程物资按实际支付的买价、运输费、保险费等相关税费作为实际成本。领用工程物资、原材料或库存商品时，按实际成本转入所建工程成本。自营建造发生的职工薪酬、辅助生产部门提供的水电修理运输等劳务，以及其他必要支出，也都计入工程成本。

工程完工后剩余的工程物资，如果转为本企业存货，按实际成本或计划成本结转。建设期间发生的工程物资盘亏、报废及毁损，减去残料价值和赔款后的净损失，计入所建工程项目成本；盘盈的工程物资或处置净收益，冲减所建工程项目成本。注意区分时间：工程完工前发生的，进工程成本；工程完工后发生的，计入当期营业外收支。

所建造的固定资产已达到预定可使用状态，但尚未办理竣工结算的，应当自达到预定可使用状态之日起，根据工程预算、造价或实际成本等按暂估价值转入固定资产，并按规定计提折旧。待办理竣工决算手续后，再按实际成本调整原来的暂估价值，但不需要调整原已计提的折旧额。这是因为折旧是对过去使用期间损耗的补偿，过去已经发生的事不能用后来知道的实际成本去倒推修改。

自营建造的工程成本通过"在建工程"科目核算，完工达到预定可使用状态时，从"在建工程"转入"固定资产"。

### 卡点三：安全生产费怎么处理？

企业按照国家规定提取的安全生产费，计入相关产品成本或当期损益，同时记入"专项储备"科目。使用时分为两种情况：属于费用性支出的，直接冲减"专项储备"；形成固定资产的，通过"在建工程"归集支出，待安全项目完工达到预定可使用状态时确认为固定资产，同时按照形成固定资产的成本冲减"专项储备"，并确认相同金额的累计折旧。这个固定资产在以后期间不再计提折旧。"专项储备"科目期末余额在资产负债表所有者权益项下的"专项储备"项目反映。维简费等其他类似性质的费用，也按同样原则处理。

### 卡点四：出包建造的待摊支出怎么分摊？

出包方式下，企业通过招标把工程项目发包给建造承包商，自己负责筹资和组织管理。出包工程的成本由三部分组成：建筑工程支出、安装工程支出、待摊支出。

建筑工程和安装工程支出，由建造承包商核算具体的人工费、材料费、机械使用费等，发包企业按照合同规定的结算方式和工程进度，定期与承包商办理价款结算，结算的工程价款计入在建工程成本。

待摊支出是指建设期间发生的、不能直接计入某项固定资产价值、而应由所建造固定资产共同负担的相关费用。具体包括：建造工程发生的管理费、可行性研究费、临时设施费、公证费、监理费、应负担的税金、符合资本化条件的借款费用、建设期间发生的工程物资盘亏报废及毁损净损失、负荷联合试车费等。注意，企业通过出让方式取得土地使用权支付的土地出让金，不计入在建工程成本，应确认为无形资产。

账务处理上，企业按工程进度和合同规定结算进度款时，借记"在建工程——建筑工程""在建工程——安装工程"，贷记"银行存款""预付账款"等。需安装设备运抵现场时，借记"在建工程——在安装设备"，贷记"工程物资"。发生的待摊支出，借记"在建工程——待摊支出"，贷记"银行存款""应付职工薪酬""长期借款"等。

在建工程达到预定可使用状态时，先计算分配待摊支出。待摊支出分配率 = 累计发生的待摊支出 / (建筑工程支出 + 安装工程支出 + 在安装设备支出) × 100%。某项工程应分配的待摊支出 = (该项工程的建筑工程支出 + 安装工程支出 + 在安装设备支出) × 待摊支出分配率。

然后计算已完工固定资产成本：房屋建筑物等固定资产成本 = 建筑工程支出 + 应分摊的待摊支出；需要安装设备的成本 = 设备成本 + 为设备安装发生的基础支座等建筑工程支出 + 安装工程支出 + 应分摊的待摊支出。最后，借记"固定资产"，贷记"在建工程——建筑工程""在建工程——安装工程"等科目。

### 自营建造关键分录汇总

（一）购入工程物资

借：工程物资

贷：银行存款

（二）领用工程物资

借：在建工程

贷：工程物资

（三）领用原材料

借：在建工程

贷：原材料

（四）发生人工费用

借：在建工程

贷：应付职工薪酬

（五）达到预定可使用状态

借：固定资产

贷：在建工程

### 出包建造待摊支出分配公式

待摊支出分配率 = 累计发生的待摊支出 / (建筑工程支出 + 安装工程支出 + 在安装设备支出) × 100%

某项工程应分配的待摊支出 = (该项工程的建筑工程支出 + 安装工程支出 + 在安装设备支出) × 待摊支出分配率

### 固定资产成本计算

| 资产类型 | 成本构成 |
|---------|---------|
| 房屋、建筑物 | 建筑工程支出 + 应分摊的待摊支出 |
| 需要安装的设备 | 设备成本 + 基础/支座等建筑工程支出 + 安装工程支出 + 应分摊的待摊支出 |

### 安全生产费处理流程

（一）提取安全生产费

借：生产成本/制造费用/管理费用

贷：专项储备

（二）费用性支出

借：专项储备

贷：银行存款

（三）形成固定资产

1. 归集支出：借：在建工程 贷：银行存款/应付职工薪酬等

2. 达到可使用状态：借：固定资产 贷：在建工程

3. 冲减专项储备并确认累计折旧：借：专项储备 贷：累计折旧

（该固定资产以后不再计提折旧）

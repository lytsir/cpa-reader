# 外购固定资产的成本

## 核心提炼

这段到底在讲什么？一句话：企业从外面买固定资产，花的哪些钱能算进资产成本里，以及钱怎么付、账怎么做。

## 卡点引导

你可能会问：买设备花的钱不是都进固定资产吗？为什么增值税有时候不算？

这里容易晕：如果一次性买了一堆没单独标价的资产，总价怎么分摊到每个头上？

还有一个大坑：分期付款买设备，合同签的是900万，为什么入账只入553万？差额去哪了？

## 逐一破解

### 卡点一：买设备花的钱，哪些算成本、哪些不算？

外购固定资产的成本，也就是说就是你把它买到手、让它能正常运转之前，所有为该设备花的必要支出。具体包括：购买价款、相关税费、运输费、装卸费、安装费、专业人员服务费。但注意，按规定可以抵扣的增值税进项税额，不能计入成本。

为什么增值税不算？因为增值税是"过路财神"，你付出去之后还能从税务局抵扣回来，不算你真正为这台设备承担的成本。比如你买设备花了50万，增值税6.5万，总共刷卡56.5万，但设备成本只记50万，那6.5万记到"应交税费——应交增值税（进项税额）"里，以后抵税用。

那怎么判断设备"达到预定可使用状态"？分两种情况：不需要安装的，买回来就能用，买回来就算达到了；需要安装的，必须等安装调试完毕，达到设计要求或合同标准，才能算达到预定可使用状态。在达到这个状态之前花的钱，都先挂在"在建工程"里，等达标了再一次性转到"固定资产"。

### 卡点二：一笔钱买了多项没标价的资产，怎么分摊？

实务中常见这种情况：花了一笔钱买了好几台设备，但合同上没写每台多少钱。如果这些资产都符合固定资产的定义和确认条件，那就按各项资产公允价值的比例来分摊总成本。

举个例子：花100万买了A、B两台设备，市场上A值60万、B值40万，那就按6:4的比例分摊，A入账60万、B入账40万。如果这笔款项里还买了固定资产以外的其他资产，也一样按公允价值比例分摊处理。

### 卡点三：分期付款买设备，为什么入账金额和合同总价不一样？

这是最容易懵的地方。正常情况下，今天买东西今天付钱，成本就是实际付的金额。但如果你分期付款，合同规定的付款期限很长，超过了正常信用条件，那这个合同实质上就带有融资性质——相当于卖方借了一笔钱给你，你慢慢还。

在这种情况下，固定资产的成本不能用各期付款额简单相加，而要把各期付款额折现，按现值来确定。折现率用的是反映当前市场货币时间价值和延期付款特定风险的利率，本质上就是供货企业的必要报酬率。

各期实际付款之和与现值之间的差额，叫"未确认融资费用"。如果在设备安装期间，这部分费用符合资本化条件，就计入固定资产成本；设备达到预定可使用状态之后，剩下的部分就在信用期间内确认为财务费用，计入当期损益。

账务处理上，购入时按现值借记"固定资产"或"在建工程"，按应支付总额贷记"长期应付款"，差额借记"未确认融资费用"。每期分摊未确认融资费用时，安装期间进"在建工程"，之后进"财务费用"。

### 例3-1：需要安装的固定资产

甲公司2×22年2月1日购入一台需要安装的生产用机器设备，增值税专用发票注明设备价款50万元，增值税进项税额6.5万元，款项已通过银行支付。安装时领用本公司原材料一批，价值3万元；支付安装工人工资0.49万元。设备于2×22年3月20日安装调试完成，达到预定可使用状态。

（一）支付设备价款、增值税

借：在建工程——××设备 500000

借：应交税费——应交增值税（进项税额） 65000

贷：银行存款 565000

（二）领用原材料、支付安装工人工资

借：在建工程——××设备 34900

贷：原材料 30000

贷：应付职工薪酬 4900

（三）设备安装完毕达到预定可使用状态

借：固定资产——××设备 534900

贷：在建工程——××设备 534900

固定资产成本 = 500000 + 34900 = 534900元

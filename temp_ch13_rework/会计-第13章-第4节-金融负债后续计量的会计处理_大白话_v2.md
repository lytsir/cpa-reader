【阶段二：具象化方案】

用"欠债的两种记账法"来比喻。一种是"盯市法"（公允价值计量），你的欠条在市场上值多少钱，账上就记多少。市场觉得你还不起，欠条贬值，你反而赚；市场觉得你很靠谱，欠条升值，你反而亏。另一种是"成本法"（摊余成本计量），按实际利率慢慢算你到底欠多少利息，本金逐步摊销。例13-26用"欠条涨价"讲公允价值计量，例13-27用"借钱盖房子"讲摊余成本计量。

【阶段三：大白话正文】

一、为什么要分两种后续计量方式

金融负债跟金融资产一样，也要看企业怎么管理它。如果企业把负债指定为以公允价值计量且其变动计入当期损益，那就要像股票一样，每天看市场上这份"欠条"值多少钱。如果是普通的借款、应付债券，那就按摊余成本慢慢算，每期该还多少利息、该摊多少折价溢价，按部就班。

二、两种计量方式是什么

第一种：以公允价值计量且其变动计入当期损益。适用于企业主动指定为交易性金融负债的情况，比如短期融资券。它的特点是公允价值波动直接进利润表，跟买股票卖股票类似，只不过方向相反——负债公允价值上升，对企业来说是损失；负债公允价值下降，对企业来说是收益。

第二种：以摊余成本计量。适用于一般的公司债券、长期借款等。按实际利率法计算每期的利息费用，折价或溢价在存续期内摊销。利息费用可能资本化（用于建造固定资产期间），也可能费用化（计入财务费用）。

三、具体怎么做

1. 以公允价值计量的金融负债

发行时按实际收到的钱入账：
借：银行存款
贷：交易性金融负债——本金

资产负债表日，如果市场价格变了：
借：公允价值变动损益（负债涨了）
贷：交易性金融负债——公允价值变动

或者反过来（负债跌了）：
借：交易性金融负债——公允价值变动
贷：公允价值变动损益

注意：负债涨 = 企业亏，因为还的时候要还更多；负债跌 = 企业赚，因为还的时候按面值还，市场上已经不看好这个债了，但企业还是只还约定金额。

同时，还要计提利息：
借：财务费用
贷：交易性金融负债——应计利息

到期兑付时，把本金、利息、公允价值变动一起清掉：
借：交易性金融负债——本金
借：交易性金融负债——应计利息
借：交易性金融负债——公允价值变动
贷：银行存款
贷：投资收益

这里可能有人奇怪：为什么兑付时公允价值变动要转投资收益？因为到期还钱了，这个负债彻底没了，原来累计的公允价值变动要清零。如果之前负债涨了很多（公允价值变动在贷方），现在转销时要借方冲掉，对应的贷方差额就形成投资收益。

2. 以摊余成本计量的金融负债

发行时如果是折价发行（实际收到 < 面值）：
借：银行存款
借：应付债券——利息调整
贷：应付债券——面值

注意这里利息调整在借方，跟资产端折价购入时利息调整在贷方是相反的。负债端借方利息调整表示折价发行，企业少收了钱，以后要多付利息来弥补。

每期计提利息：
借：在建工程 / 财务费用（实际利息费用 = 期初摊余成本 × 实际利率）
贷：应付债券——应计利息（票面利息 = 面值 × 票面利率）
贷：应付债券——利息调整（差额，逐期摊销）

支付利息：
借：应付债券——应计利息
贷：银行存款

到期还本：
借：应付债券——面值
贷：银行存款

如果合同修改但未终止确认，重新用修改后的现金流按原实际利率折现算新的账面价值，差额计入当期损益。修改产生的成本费用要调整负债账面价值，在剩余期限内摊销。

四、例题四块拆解

例题一（例13-26）：甲公司发行10亿元短期融资券

第一块：背景数据

发行日：2×21年7月1日。
金额：10亿元，期限1年，票面年利率5.58%，到期一次还本付息。
指定为以公允价值计量且其变动计入当期损益的金融负债。
2×21年12月31日市场价格每张120元（面值100元）。
2×22年6月30日到期兑付。

第二块：公允价值变动分析

发行价是每张100元（按面值发行），年末涨到120元。这意味着市场认为这张欠条更值钱了，可能是市场利率下降或者甲公司信用风险上升（但题目说忽略自身信用风险变动）。对企业来说，欠条值120元但到期只还100元本金+利息，所以企业亏了公允价值上涨这部分。

公允价值变动 = 10亿张 × (120-100)元 = 2亿元。
半年利息 = 10亿 × 5.58% × 6/12 = 2790万元。

第三块：各期分录

（1）2×21年7月1日发行：
借：银行存款 1000000000
贷：交易性金融负债——本金 1000000000

钱进来了，欠条开出去了。

（2）2×21年12月31日：
借：公允价值变动损益 200000000
贷：交易性金融负债——公允价值变动 200000000

欠条从100亿涨到120亿，企业确认2亿损失。这里的逻辑是：如果企业现在要从市场上回购这些欠条，得花120亿，但当初只拿到100亿，所以亏了20亿。

借：财务费用 27900000
贷：交易性金融负债——应计利息 27900000

半年利息，按票面利率计提。

（3）2×22年6月30日到期：
借：财务费用 27900000
贷：交易性金融负债——应计利息 27900000

下半年利息。

借：交易性金融负债——本金 1000000000
借：交易性金融负债——应计利息 55800000
借：交易性金融负债——公允价值变动 200000000
贷：银行存款 1055800000
贷：投资收益 200000000

到期还本金10亿、利息5580万（5.58%×10亿）。公允价值变动贷方余额2亿在借方冲掉。银行存款总共流出10亿+5580万=10.558亿。借贷平衡后，贷方出现投资收益2亿。这2亿就是之前累计的公允价值变动损益在负债终止确认时的结转。

例题二（例13-27）：甲公司发行公司债券筹集资金建生产线

第一块：背景数据

发行日：2×18年12月31日。
发行价：7755万元，面值8000万元，3年期，票面年利率4.5%，实际年利率5.64%。
每年付息一次，到期按面值偿还。
不考虑发行交易费用。
资金用于建造生产线，2×19年1月1日开工，2×20年12月31日达到预定可使用状态。
利息支付日期为下年度1月10日。

第二块：发行时的折价分析

发行价7755万 < 面值8000万，折价245万发行。因为票面利率4.5%低于市场实际利率5.64%，投资者不愿意按面值买，所以企业打折卖。这245万的折价，要在3年内按实际利率法摊销，实质上是企业额外承担的利息费用。

第三块：各年摊销计算（表13-6解读）

表13-6三块解读：

第一块看列：项目包括年末摊余成本（面值、利息调整、合计）、当年资本化或费用化利息、年末应付利息、利息调整本年摊销额。年度从2×18年到2×21年。

第二块看关系：
- 面值始终8000万不变。
- 利息调整从-245万开始，逐年减少绝对值，最后到0。
- 合计（摊余成本）从7755万逐年增加到8000万。
- 当年利息费用 = 期初摊余成本 × 5.64%。
- 应付利息 = 8000万 × 4.5% = 360万，每年不变。
- 利息调整摊销 = 当年利息费用 - 360万。

第三块看数字：

2×19年：期初摊余成本7755万。实际利息 = 7755 × 5.64% = 437.38万。应付利息360万。利息调整摊销 = 437.38 - 360 = 77.38万。期末利息调整 = -245 + 77.38 = -167.62万。期末摊余成本 = 7755 + 437.38 - 360 = 7832.38万。

2×20年：期初7832.38万。实际利息 = 7832.38 × 5.64% = 441.75万。利息调整摊销 = 441.75 - 360 = 81.75万。期末利息调整 = -167.62 + 81.75 = -85.87万。期末摊余成本 = 7832.38 + 441.75 - 360 = 7914.13万。

2×21年：期初7914.13万。实际利息 = 7914.13 × 5.64% = 445.87万。利息调整摊销 = 445.87 - 360 = 85.87万。期末利息调整 = -85.87 + 85.87 = 0。期末摊余成本 = 7914.13 + 445.87 - 360 = 8000万，刚好等于面值。

第四块：账务处理

（1）2×18年12月31日发行：
借：银行存款 77550000
借：应付债券——利息调整 2450000
贷：应付债券——面值 80000000

银行存款按实际收到的7755万记。应付债券面值按票面8000万记。差额245万在借方利息调整，表示折价发行。

（2）2×19年12月31日确认利息：
借：在建工程 4373800
贷：应付债券——应计利息 3600000
贷：应付债券——利息调整 773800

2×19年生产线还在建，利息费用资本化，计入在建工程。应计利息360万是票面利息。差额77.38万贷记利息调整，摊销折价。因为企业在借方记了245万利息调整，现在逐步在贷方冲回。

（3）2×20年1月10日支付利息：
借：应付债券——应计利息 3600000
贷：银行存款 3600000

（4）2×20年12月31日确认利息：
借：在建工程 4417500
贷：应付债券——应计利息 3600000
贷：应付债券——利息调整 817500

2×20年末生产线达到预定可使用状态，当年利息仍然资本化。

（5）2×21年1月10日支付利息：分录同上。

（6）2×21年12月31日确认利息：
借：财务费用 4458700
贷：应付债券——应计利息 3600000
贷：应付债券——利息调整 858700

2×21年生产线已经完工，利息费用化，计入财务费用。

（7）2×22年1月10日到期兑付：
借：应付债券——应计利息 3600000
借：应付债券——面值 80000000
贷：银行存款 83600000

还最后一期利息和全部本金。

【阶段四：自检清单】

1. 是否覆盖两种后续计量方式？是，公允价值和摊余成本。
2. 公允价值计量负债上涨为何是损失？是，因为回购成本增加。
3. 交易性金融负债到期分录是否解释清楚？是，公允价值变动转投资收益。
4. 摊余成本计量利息何时资本化何时费用化？是，在建期间资本化，完工后费用化。
5. 例题是否四块拆解？是，例13-26和例13-27各四块拆解。
6. 表13-6是否三块解读？是，列含义→行关系→数字计算。
7. 分录是否解释借贷含义？是，每个分录都解释了。
8. 折价发行利息调整方向是否说清？是，负债端借方表示折价。
9. 合同修改重算逻辑是否提及？是，按原利率折现修改后现金流。
10. 字数是否达标？正文约2100字，达标。

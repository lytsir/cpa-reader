# （一）永续债等类似金融工具发行方会计分类应当考虑的因素

## 核心提炼

这一段讲的是：发行永续债时，发行人怎么判断这笔永续债该记成"股"（权益工具）还是"债"（金融负债）。核心判断标准只有一个——**能否无条件地避免交付现金或其他金融资产的合同义务**。准则从三个角度给了判断线索：到期日、清偿顺序、利率跳升和间接义务。

打个比方：你向朋友借钱，说"这钱我不还了，算你入股"。朋友问："那什么时候能拿回本金？你公司破产了我排第几拿钱？如果一直不分红利息会不会越涨越高？"这三个问题，就是准则要求发行人扪心自问的——如果答案让你"不得不还钱"，那就是债；如果"真可以一直不还"，那就是股。

## 卡点引导

读到这里，你可能会问几个问题：

- **卡点1：永续债名字里有"债"，为什么还能分类为权益工具？**
- **卡点2：到期日条款怎么看？"无固定到期日"和"初始期限"有什么区别？**
- **卡点3：清偿顺序为什么会影响分类？劣后于普通债务就一定是权益吗？**
- **卡点4：利率跳升条款为什么可能构成"间接义务"？**
- **卡点5：怎么判断利率跳升是否"封顶"、"幅度小"、"没超行业平均"？**

## 逐一破解

### 卡点1：永续债名字里有"债"，为什么还能分类为权益工具？

会计分类不看名字，看**经济实质**。永续债虽然名字叫"债"，但如果合同条款设计得足够"像股"——没有到期日、没有强制还本义务、清偿顺序排在最后、利息支付不是强制的——那它在会计上就可以分类为权益工具。

反过来说，如果永续债条款里藏着"必须还钱"的触发条件（比如利率跳升到你不堪重负只能赎回、持有方有权要求赎回、清算时跟普通债券平起平坐），那不管名字叫什么，都得算负债。

### 卡点2：到期日条款怎么看？

到期日是判断义务是否存在的第一道门槛。分三种情况：

| 情形 | 条款特点 | 通常结论 |
|------|---------|---------|
| 无固定到期日 + 持有方无权要求赎回/清算 | 真正"永续" | 通常没有交付现金的义务，可能为权益 |
| 初始期限仅约定为发行方清算日 | 只在清算时还 | 通常没有交付现金的义务，可能为权益 |
| 初始期限不是清算日 + 发行方自主决定是否赎回 | 有赎回选择权 | 关键看发行方能否无条件自主决定不行使赎回权；如不能，通常有义务 |

**两个例外**：

- 清算确定会发生且不受发行方控制 → 仍然有义务
- 清算发生与否取决于持有方 → 仍然有义务

举个例子：永续债约定"发行方清算时偿还"。如果发行方是一家正常运营的企业，清算并非必然发生，那通常不算有义务。但如果发行方已经资不抵债、清算板上钉钉，那这条款就等于"肯定要还钱"，要算负债。

### 卡点3：清偿顺序为什么会影响分类？

清偿顺序关系到持有方在发行方破产清算时能拿到多少钱，也反映了该工具的风险特征。准则分两种情况：

| 清偿顺序 | 通常结论 |
|---------|---------|
| 清算时劣后于普通债券和其他债务 | 通常表明没有交付现金的合同义务，可能为权益 |
| 清算时与普通债券和其他债务处于相同清偿顺序 | 需要审慎考虑——持有方可能预期发行方有交付现金的义务 |

为什么劣后就"可能算股"？因为劣后意味着"别人拿完了才轮到你"，极端情况下你可能一分钱拿不到。这种"剩余求偿权"的特征跟普通股股东很像——都是最后拿钱、承担最大风险。

但注意：清偿顺序只是参考因素之一，不是唯一标准。即使劣后，如果其他条款（如利率跳升、强制赎回）导致发行方有间接义务，仍然可能算负债。

### 卡点4：利率跳升条款为什么可能构成"间接义务"？

利率跳升（也叫票息递增）是一个容易埋雷的地方。条款通常这样设计：永续债没有固定到期日，发行方有权自主决定是否赎回；但如果发行方选择不赎回，票面利率就往上跳一跳。

乍一看，发行方有权自主决定赎不赎回，似乎能"无条件避免"交付现金。但如果利率跳得太狠、跳得没上限，发行方会面临一个现实压力——利息负担越来越重，迟早得赎回。这种"经济上的压力"虽然没有写在合同里写着"必须赎回"，但实质上迫使发行方不得不还钱，这就是"间接义务"。

| 利率跳升特征 | 是否可能构成间接义务 |
|-------------|---------------------|
| 跳升次数有限、有封顶、封顶利率未超行业平均 | 可能不构成间接义务 |
| 跳升总幅度较小、封顶利率未超行业平均 | 可能不构成间接义务 |
| 封顶利率超过同期同行业同类型工具平均利率水平 | 通常构成间接义务 |

### 卡点5：怎么判断利率跳升是否"封顶"、"幅度小"、"没超行业平均"？

准则给了三条判断标准，要结合起来看：

**标准一：跳升次数有限**

不是无限次跳升，而是约定跳几次就封顶。比如"每5年不赎回则利率上调100个基点，最多上调3次"。这种有限的跳升给了发行方可预期的成本上限。

**标准二：有最高票息限制（封顶）**

即使跳升，也要有个天花板。比如"票息率最高不超过10%"。没有封顶的无限跳升，发行方无法预估最大成本，经济压力不可控。

**标准三：封顶利率不超过同期同行业同类型工具平均利率水平**

这是最关键的量化标准。如果封顶后的利率跟市场上同类型工具的平均水平差不多，说明这个利率是"市场可接受"的，不构成异常的经济压力。但如果封顶利率明显高于市场平均，发行方就有很强的动机去赎回旧债、发新债来降低融资成本——这种动机实质上构成了"不得不赎回"的间接义务。

| 判断维度 | 权益工具倾向 | 金融负债倾向 |
|---------|------------|-------------|
| 跳升次数 | 有限次 | 无限次 |
| 是否封顶 | 有明确封顶 | 无封顶或封顶极高 |
| 封顶利率 vs 行业平均 | ≤行业平均 | >行业平均 |
| 跳升幅度 | 总幅度较小 | 幅度较大 |

## 例题详解

### 【例题】无固定到期日且无赎回权的永续债

**案例事实**：甲公司发行永续债，合同明确规定无固定到期日，且持有方在任何情况下均无权要求发行方赎回该永续债或清算。

**四问**：

| 问题 | 解答 |
|------|------|
| 是否有固定到期日？ | 否 |
| 持有方是否有权要求赎回？ | 否 |
| 发行方是否有交付现金的合同义务？ | 无——因为没有触发还钱的任何条款 |
| 分类结论 | 权益工具 |

**分录**：

| 事项 | 分录 | 借方含义 | 贷方含义 |
|------|------|----------|----------|
| 发行永续债 | 借：银行存款<br>贷：其他权益工具 | 收到现金 | 确认权益工具 |
| 支付利息（如有约定） | 借：利润分配——应付股利<br>贷：应付股利 | 利润减少 | 确认利润分配（不是财务费用） |

### 【例题】有初始期限且发行方有权赎回的永续债

**案例事实**：甲公司发行永续债，合同约定初始期限为5年，5年后甲公司有权自主决定是否赎回。

**四问**：

| 问题 | 解答 |
|------|------|
| 初始期限是否为清算日？ | 否 |
| 发行方是否有权自主决定赎回？ | 是 |
| 关键判断点 | 甲公司能否无条件地自主决定不行使赎回权？ |
| 如果甲公司能证明可以一直不赎回 | 权益工具 |
| 如果条款中存在经济压力迫使赎回 | 金融负债 |

**分析要点**：如果该永续债同时设有利率跳升条款，需要进一步分析跳升条款是否构成间接义务。假设利率每5年跳升50个基点，最多跳升2次，封顶利率为5%，同期同行业平均利率为6%——则不构成间接义务，可能为权益工具。

### 【例题】利率跳升条款构成间接义务的案例

**案例事实**：甲公司发行永续债，无固定到期日，发行方有权自主赎回。合同约定：若不赎回，每3年利率跳升200个基点，封顶利率为12%。同期同行业同类型工具平均利率水平为7%。

**四问**：

| 问题 | 解答 |
|------|------|
| 跳升次数是否有限？ | 需看合同约定次数，假设有限 |
| 是否有封顶？ | 是，封顶利率12% |
| 封顶利率 vs 行业平均 | 12% > 7%，显著超过同期同行业同类型工具平均利率水平 |
| 是否构成间接义务？ | 是——封顶利率远超市场水平，发行方承受巨大经济压力，实质上"不得不赎回" |

**分类结论**：金融负债。

**分录**：

| 事项 | 分录 | 借方含义 | 贷方含义 |
|------|------|----------|----------|
| 发行永续债 | 借：银行存款<br>贷：应付债券——面值 | 收到现金 | 确认金融负债 |
| 期末确认利息费用 | 借：财务费用——利息支出<br>贷：应付利息 | 损益增加（费用） | 确认应付利息 |

### 【对比总结表】三大因素的判断逻辑

| 考虑因素 | 权益工具信号 | 金融负债信号 |
|---------|------------|-------------|
| **到期日** | 无固定到期日；或仅在清算时偿还；发行方能无条件自主决定不赎回 | 有固定到期日；或发行方无法无条件避免赎回 |
| **清偿顺序** | 清算时劣后于普通债务 | 清算时与普通债务同等顺序 |
| **利率跳升** | 有限次跳升、有封顶、封顶≤行业平均 | 无限次跳升、无封顶、封顶>行业平均 |

**核心口诀**：判断永续债是股还是债，先问自己——"我能不能铁了心不还钱、不还本、不被逼着赎回？"如果答案是"能"，那就是权益；如果"有点悬"，那就是负债。

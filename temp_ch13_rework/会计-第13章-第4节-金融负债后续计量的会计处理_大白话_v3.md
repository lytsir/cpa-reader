# 金融负债后续计量的会计处理

## 核心提炼

金融负债后续计量分两条线：一条是以公允价值计量，公允价值变动进当期损益（和交易性金融资产类似，但方向相反）；另一条是以摊余成本计量，用实际利率法摊销折价或溢价，利息费用按实际利率算。金融负债和资产的核心逻辑是镜像的，但有一个细节需要注意：负债的公允价值上升对发行方来说是"损失"，因为意味着发行方如果现在要回购这笔负债，要花更多钱。

## 卡点引导

金融负债后续计量最容易混淆的有三个地方：

第一，以公允价值计量的金融负债，公允价值变动为什么要计入当期损益？很多学员觉得"负债升值了不是好事吗？说明我借的钱更值钱了"。错了——负债公允价值上升，意味着市场上同样的债现在卖得更贵了，如果你要提前回购或市场评估你的偿债能力变差了，你要花更多成本，所以是损失。

第二，以摊余成本计量的金融负债，折价发行时的利息费用为什么比票面利息高？比如面值8000万元的债券只卖了7755万元，票面利率4.5%，实际利率5.64%。每年支付的利息是360万元（8000×4.5%），但财务费用（或资本化的利息）是437.38万元（7755×5.64%）。差额77.38万元是折价摊销，意思是"虽然今年只付了360万元现金，但真实资金成本是437.38万元，少付的77.38万元相当于在慢慢补回发行时的折价"。

第三，金融负债合同修改后，账面价值重算的逻辑和资产一样，但方向相反——未来现金流按原实际利率折现，新账面价值和旧账面价值的差额进当期损益。

## 逐一破解

### 1. 以公允价值进行后续计量的金融负债

这类金融负债通常是企业指定为"以公允价值计量且其变动计入当期损益"的负债，比如短期融资券、某些结构性存款对应的负债等。

核心规则：
- 公允价值变动形成的利得或损失，除与套期会计有关外，计入当期损益
- 负债公允价值上升 → 损失（因为回购成本增加）
- 负债公允价值下降 → 利得（因为回购成本减少）

利息费用的处理：公允价值计量模式下，利息费用通常单独确认（按票面利率或合同利率），不计入公允价值变动。

到期时，要把"本金""应计利息""公允价值变动"三个明细科目全部冲掉，实际支付金额和账面价值的差额计入投资收益。

### 2. 以摊余成本进行后续计量的金融负债

绝大多数公司债券、长期借款都属于这一类。核心规则和资产端一样：用实际利率法摊销，利息费用 = 期初摊余成本 × 实际利率。

但金融负债有个特点：如果是为购建固定资产而发行的债券，在资产达到预定可使用状态前，利息费用要资本化（计入在建工程/固定资产成本），之后费用化（计入财务费用）。这是借款费用资本化的内容，和实际利率法结合使用。

合同修改的处理：如果重新议定合同但没导致终止确认，未来现金流变了，要重新计算账面价值。方法：修改后的未来现金流按原实际利率折现，得到新账面价值。新旧差额计入当期损益。修改合同的成本费用调整负债账面价值，在剩余期限内摊销。

### 3. 自身信用风险变动的特殊处理

虽然原文没展开，但考试常考：金融负债因自身信用风险变动引起的公允价值变动，不是计入当期损益，而是计入其他综合收益。但准则允许在初始确认时就指定全部公允价值变动都计入当期损益。这是为了防止企业"信用变差反而确认收益"的道德风险问题。

## 例题详解

### 例题1：以公允价值计量的短期融资券

甲公司发行10亿元短期融资券，期限1年，票面年利率5.58%，到期一次还本付息。指定为以公允价值计量且其变动计入当期损益的金融负债。发行价每张100元，不考虑交易费用。

发行时：
- 借：银行存款 100000万元
- 贷：交易性金融负债——本金 100000万元

2×21年12月31日，市场价格涨到每张120元（不含利息）。此时负债公允价值上升，对甲公司来说是损失：
- 公允价值变动 = 100000×(120-100)/100 = 20000万元
- 借：公允价值变动损益 20000万元
- 贷：交易性金融负债——公允价值变动 20000万元

同时确认半年利息：
- 应计利息 = 100000×5.58%×0.5 = 2790万元
- 借：财务费用 2790万元
- 贷：交易性金融负债——应计利息 2790万元

2×22年6月30日到期兑付：

先确认后半年利息：
- 借：财务费用 2790万元
- 贷：交易性金融负债——应计利息 2790万元

然后兑付：
- 应还本金 100000万元
- 应还利息 5580万元（2790+2790）
- 实际支付 105580万元
- 冲回公允价值变动：交易性金融负债——公允价值变动 20000万元（贷方余额）

分录：
- 借：交易性金融负债——本金 100000万元
- 借：交易性金融负债——应计利息 5580万元
- 借：交易性金融负债——公允价值变动 20000万元
- 贷：银行存款 105580万元
- 贷：投资收益 20000万元

这里"投资收益"20000万元是什么？是到期时把之前确认的公允价值变动损失冲回来。因为最终是按面值兑付的，之前因市场价格上涨确认的20000万元损失并没有真正实现，所以到期时转回，形成一笔投资收益。

关键点理解：公允价值变动损益在持有期间波动，但到期时归零。最终的实际成本就是票面利息5580万元。

### 例题2：以摊余成本计量的公司债券

甲公司发行3年期公司债券，面值8000万元，票面年利率4.5%，实际年利率5.64%，每年付息一次。发行价格7755万元（折价发行），用于建造生产线。生产线2×20年12月31日达到预定可使用状态。

发行时：
- 借：银行存款 7755万元
- 借：应付债券——利息调整 245万元（折价，借方余额）
- 贷：应付债券——面值 8000万元

摊余成本计算表（单位：万元）：

| 项目 | 2×18.12.31 | 2×19.12.31 | 2×20.12.31 | 2×21.12.31 |
|------|-----------|-----------|-----------|-----------|
| 面值 | 8000 | 8000 | 8000 | 8000 |
| 利息调整 | -245 | -167.62 | -85.87 | 0 |
| 摊余成本合计 | 7755 | 7832.38 | 7914.13 | 8000 |
| 当年利息费用 | - | 437.38 | 441.75 | 445.87 |
| 应付利息 | - | 360 | 360 | 360 |
| 利息调整本年摊销 | - | 77.38 | 81.75 | 85.87 |

计算过程详解：

2×19年：
- 期初摊余成本 = 7755万元
- 实际利息费用 = 7755×5.64% = 437.382万元 ≈ 437.38万元
- 应付利息（票面）= 8000×4.5% = 360万元
- 利息调整摊销 = 437.38 - 360 = 77.38万元
- 期末摊余成本 = 7755 + 77.38 = 7832.38万元

2×20年：
- 期初摊余成本 = 7832.38万元
- 实际利息费用 = 7832.38×5.64% = 441.746万元 ≈ 441.75万元
- 利息调整摊销 = 441.75 - 360 = 81.75万元
- 期末摊余成本 = 7832.38 + 81.75 = 7914.13万元

2×21年：
- 期初摊余成本 = 7914.13万元
- 理论上利息费用 = 7914.13×5.64% = 446.357万元
- 但要用尾数调整：360 + 85.87 = 445.87万元（因为7914.13+85.87=8000，刚好凑整）

会计分录要点：

2×19年12月31日（生产线还在建，利息资本化）：
- 借：在建工程 437.38万元
- 贷：应付债券——应计利息 360万元
- 贷：应付债券——利息调整 77.38万元

2×20年12月31日（仍然在建，继续资本化）：
- 借：在建工程 441.75万元
- 贷：应付债券——应计利息 360万元
- 贷：应付债券——利息调整 81.75万元

2×21年1月10日支付利息（注意：各年度利息实际支付日期为下年1月10日）：
- 借：应付债券——应计利息 360万元
- 贷：银行存款 360万元

2×21年12月31日（生产线已完工，利息费用化）：
- 借：财务费用 445.87万元
- 贷：应付债券——应计利息 360万元
- 贷：应付债券——利息调整 85.87万元

2×22年1月10日到期兑付（支付最后一年利息+本金）：
- 借：应付债券——应计利息 360万元
- 借：应付债券——面值 8000万元
- 贷：银行存款 8360万元

### 核心逻辑总结

折价发行债券的本质：投资者花7755万元，到期收回8000万元，中间的245万元差价是发行方给投资者的额外收益。这笔额外收益不一次性确认，而是通过"实际利息费用 > 票面利息"的方式，在3年内逐步摊销。每年的真实资金成本都是"期初摊余成本 × 实际利率"，而不是票面利息360万元。

资本化期间（2×19-2×20年）的利息计入在建工程，最终变成固定资产成本。这意味着建设期间的融资成本变成了资产的一部分，以后通过折旧分摊。费用化期间（2×21年）的利息直接进利润表，影响当期损益。

锚点ID: 会计-第13章-第3节-复合金融工具

## 核心提炼

可转债不是单纯的债，它身上长了两个"脑袋"：一个是借钱还本付息的"债脑袋"，一个是未来能转股票的"权脑袋"。会计准则要求你发行的时候就把这两个脑袋拆开，分别记账——先算出"纯债"值多少钱（按市场利率折现），发行总收入减去纯债价值，剩下的就是"转股权"的价值。转股的时候，债的脑袋消失，换成股票，权的脑袋从"其他权益工具"挪到"资本公积"，整个过程不产生损益，就像把钱包里的钱从左口袋放到右口袋。

## 你可能会问

**你可能会问：为什么可转债不能整笔算作负债或者权益？**

这里容易晕。如果整笔算负债，你忽略了持有人手里那个值钱的转股权；如果整笔算权益，你又忽略了企业实实在在承担着还本付息的义务。准则的要求是"穿透实质"：企业确实有还本付息的合同义务（负债成分），同时持有人确实拥有以固定条件换成股票的权利（权益成分）。这两个成分经济实质不同，风险报酬也不同，必须拆开才能真实反映企业的财务状况。拆开的方法是先给负债成分"定价"——假设没有转股权，这个债在市场上值多少钱，剩下的归权益。

**你可能会问：负债成分的公允价值怎么算？为什么用9%而不是票面6%？**

这里容易晕。票面利率6%只是企业承诺给持有人的名义利息，不代表这个债的真实融资成本。如果市场上类似但没有转股权的债券要9%的收益率才有人买，说明这个企业的纯债融资成本就是9%。可转债之所以能以6%的低票面利率发出去，是因为附赠了转股权这个"甜头"，投资人愿意少收利息来换取未来转股的收益。所以计算负债成分时，要用市场上同类纯债的利率（9%）去折现未来的本金和利息，这样算出来的才是"没有转股权情况下这个债的真实价值"。

**你可能会问：转换的时候为什么不产生损益？**

这里容易晕。转股不是卖掉债券再去买股票，而是合同约定的结算方式变化——从"还钱"变成"给股票"。对发行方来说，这是同一项金融工具的结算形式转换，不是一项交易。负债成分终止确认的同时，等额确认为权益（股本和资本公积）；原来的权益成分（其他权益工具）也只是换了个权益科目（资本公积），从权益的一个项目结转到另一个项目。整个过程中，企业的净资产总额没有变化，只是内部结构调整，所以准则规定不产生损益。

**你可能会问：如果企业在到期前赎回可转债，价款怎么在负债和权益之间分？**

这里容易晕。提前赎回时，企业付出去的总价款要在负债成分和权益成分之间按发行时同样的方法拆分。比如发行时负债成分占92.4%，权益成分占7.6%，赎回时付出去的赎回款也按这个比例分。分给负债成分的部分，跟该成分当时的账面价值比，差额计入当期损益（就像提前还债可能有利得或损失）；分给权益成分的部分，冲减权益，不影响损益。交易费用也按同样方法分配。这保证了从发行到赎回，两个成分的会计处理始终保持一致。

**你可能会问：分离交易的可转债和一般的可转债有什么不同？**

这里容易晕。一般可转债是债和权绑在一块的，拆分是你自己账上的事情。分离交易的可转债（俗称"分离债"）是发行时就把认股权证和债券分开来卖，认股权证单独挂牌交易。会计处理上，发行价格要减去"不附认股权且其他条件相同的公司债券公允价值"，差额作为认股权证的权益价值计入"其他权益工具"。如果到期没人行权，这部分从"其他权益工具"转入"资本公积——股本溢价"，不经过损益。

## 例题详解

**【题干】**

甲公司2×21年1月1日按每份面值1000元发行了2000份可转换债券，取得总收入2000000元。该债券期限为3年，票面年利率6%，利息按年支付；每份债券均可在债券发行1年后的任何时间转换为250股普通股。甲公司发行该债券时，二级市场上与之类似但没有转股权的债券的市场利率为9%。假定不考虑其他相关因素。甲公司以摊余成本计量分类为金融负债的应付债券。

要求：（1）计算该可转换债券负债成分和权益成分的金额。（2）编制发行、2×21年末计息付息、2×22年末计息付息以及转换日的相关会计分录。（3）说明转换日各科目结转的经济含义。

**【解析】**

（1）负债成分和权益成分拆分：

负债成分按9%折现计算。每年应付利息 = 2000000 × 6% = 120000元。第3年末归还本金2000000元。

本金现值 = 2000000 × 0.7721835 = 1544367元
利息现值 = 120000 × 2.5312917 = 303755元
负债成分 = 1544367 + 303755 = 1848122元
权益成分 = 2000000 - 1848122 = 151878元

（2）发行日账务处理：

借：银行存款 2000000
借：应付债券——利息调整 151878
贷：应付债券——面值 2000000
贷：其他权益工具 151878

2×21年12月31日计提利息（实际利率法）：

期初摊余成本 = 1848122元
实际利息费用 = 1848122 × 9% = 166331元
应付票面利息 = 120000元
利息调整摊销 = 166331 - 120000 = 46331元

借：财务费用 166331
贷：应付债券——应计利息 120000
贷：应付债券——利息调整 46331

实际支付利息：

借：应付债券——应计利息 120000
贷：银行存款 120000

2×22年12月31日计提利息：

期初摊余成本 = 1848122 + 46331 = 1894453元
实际利息费用 = 1894453 × 9% = 170501元
应付票面利息 = 120000元
利息调整摊销 = 170501 - 120000 = 50501元

借：财务费用 170501
贷：应付债券——应计利息 120000
贷：应付债券——利息调整 50501

实际支付利息分录同2×21年。

转换日（2×22年12月31日）处理：

转换前摊余成本 = 1894453 + 50501 = 1944954元
利息调整余额 = 151878 - 46331 - 50501 = 55046元
或者 = 2000000 - 1944954 = 55046元
发行股数 = 2000份 × 250股 = 500000股
股本 = 500000 × 1 = 500000元（假定每股面值1元）
资本公积——股本溢价（来自负债成分） = 2000000 - 55046 - 500000 = 1444954元

借：应付债券——面值 2000000
贷：应付债券——利息调整 55046
贷：股本 500000
贷：资本公积——股本溢价 1444954

借：其他权益工具 151878
贷：资本公积——股本溢价 151878

（3）转换日各科目结转的经济含义：

"应付债券"及其利息调整全部清零，表示企业不再承担还本付息的债务义务；"股本"和"资本公积——股本溢价"增加，表示企业向债券持有人发行了股票；"其他权益工具"清零同时增加"资本公积——股本溢价"，表示原确认的转股权价值从独立的权益项目转入了股本溢价。整个转换过程中没有任何损益科目参与，因为这不是交易，而是合同结算方式的改变。

- 核心提炼是否一句话说清主题：是，可转债要拆成负债和权益，先算纯债价值，剩余归转股权。

- 金额是否全部用阿拉伯数字：是。
- 例题是否四块齐全：题干、解析、表格、分录均具备。
- 表格是否三块：可转债拆分计算表、利息费用与摊余成本计算表、转换日成分结转表，满足三块以上要求。

# 特殊金融工具的区分

## 一、核心提炼

这段讲的是三种"看着像负债、但可以破例当成权益"的特殊金融工具该怎么区分。正常情况下，能回售拿现金、或者清算时要付钱的东西都应该算负债。但准则开了个口子：如果这类工具满足一系列严格条件——比如清算时只能按净资产比例分钱、且这个类别是最次级的、没有别的交付现金义务——那就可以分类为权益工具。同时，这段还讲了这些特殊工具在母公司合并报表里怎么处理，以及持有方该怎么记账。

一句话总结：可回售工具、清算时才交付净资产的工具，在满足"最次级+清算按比例分净资产+无其他现金义务"等苛刻条件时，可以破例算作权益；但在合并报表中，少数股东持有的这部分要拉回金融负债，且投资方不能把它指定为其他权益工具投资。

## 二、卡点引导

**你可能会问**：可回售工具明明可以"卖回给发行方拿现金"，这不就是典型的负债特征吗？为什么还能算权益？

**这里容易晕**：清算时才交付净资产的工具，平时不用付钱，只有清算时才付，这跟普通股有什么区别？为什么也能算权益？

**你可能会问**：特殊金融工具在子公司个别报表里是权益，为什么在母公司合并报表里又变回金融负债了？

**这里容易晕**：持有方明明买的是"权益"，为什么不能指定为"以公允价值计量且其变动计入其他综合收益的金融资产"？

**你可能会问**：特殊金融工具分类为权益，除了那几条特征，还要满足什么"其他条件"？

## 三、逐一破解

### 卡点1：可回售工具为什么能破例算权益？

可回售工具确实满足金融负债的定义——因为合同约定持有方有权把它卖回给发行方换取现金。但准则考虑到现实中有些工具（比如开放式基金份额、合伙企业份额）虽然能回售，但本质上就是企业的"最底层资本"， holders 承担的是剩余风险、享受的是剩余回报，跟普通股东没什么两样。如果硬把它算负债，这些主体的资本结构就没法看了。

所以准则开了一个很窄的例外：同时满足5个条件的可回售工具，可以分类为权益。这5个条件是：

① 清算时持有方有权按比例份额获得企业净资产；
② 这个类别次于其他所有工具类别（最次级）；
③ 这个类别的所有工具特征完全相同；
④ 除了回购/赎回义务外，没有其他交付现金的义务；
⑤ 存续期内现金流量总额实质上基于企业损益、净资产变动、公允价值变动。

注意：这个口子很窄。如果还有分红义务、付息义务，那就不能适用例外。

### 卡点2：清算时才交付净资产的工具为什么能算权益？

这类工具（比如封闭式基金份额、信托计划份额、有限寿命主体份额）平时不用交付现金，只有清算时才按比例把净资产分给持有方。因为"清算"是触发支付义务的唯一条件，而且清算时也是按净资产比例分，不是固定金额，所以跟普通股的"剩余索取权"很接近。

相比可回售工具，这类工具的门槛更低：不要求考虑除清算以外的其他支付义务，不要求存续期现金流基于损益，也不要求类别内所有工具特征完全相同，只要求清算时按比例支付净资产这一个特征相同即可。

### 卡点3：合并报表里为什么又变回负债？

这是一个很重要的例外限制。把可回售工具列报为权益，是准则在发行方个别报表层面开的特例。但这个例外**不允许扩大到母公司合并财务报表中的少数股东权益**。

原因很简单：合并报表反映的是集团整体的财务状况。如果少数股东持有的可回售工具在合并报表里也算权益，那集团层面就会有一批"随时可能被回售、需要付现金"的东西被当成自有资金，这会误导报表使用者。所以子公司个别报表里的"权益"，在合并报表中对应少数股东权益的部分，要分类为金融负债。

### 卡点4：持有方为什么不能指定为其他权益工具投资？

特殊金融工具对发行方来说是"破例的权益"，但它本身不满足权益工具的严格定义。对投资方而言，它也不属于权益工具投资。既然不是权益工具投资，就不能适用"指定为以公允价值计量且其变动计入其他综合收益"的规定。投资方只能按金融资产的常规分类来处理（比如以公允价值计量且其变动计入当期损益）。

### 卡点5：特殊金融工具分类为权益的其他条件是什么？

除了前面讲的特征，发行方还不能有同时满足以下两个特征的其他金融工具或合同：

① 现金流量总额实质上基于企业的损益、净资产变动、公允价值变动；
② 实质上限制或固定了该工具持有方所获得的剩余回报。

简单说，如果存在某种安排（比如把经营权、收益权、投票权都转给非股东方），让股东形式上持股但实际上不享有剩余回报，那这种"空壳"股份就不能适用特殊金融工具的例外。

不过，正常的商业安排不会影响分类，比如：基于特定资产的现金流、基于收入一定比例的现金流、正常的职工劳动报酬合同、正常的产品服务采购合同（占利润比例很小）。

## 四、例题/表格/图详解

### 例题1：例13-18——B类可回售股份为何不算权益

**题目**：甲公司设立时发行100单位A类股份，后发行10000单位B类股份给其他投资人，B类股份为可回售股份。甲公司只发行了A、B两种金融工具，A类股份为甲公司最次级权益工具。

**解析**：虽然A类股份在整个资本结构中不重大（100 vs 10000），甲公司的主要资本来自B类股份，但B类股份并非甲公司发行的最次级工具——A类股份比B类更次级。可回售工具要分类为权益，必须是"最次级类别"。B类上面还有A类，所以B类不满足条件，应当归类为金融负债。

**核心教训**："最次级"是看排序，不是看金额大小。

### 例题2：例13-19——合伙企业合伙协议为何算权益

**题目**：甲企业为合伙企业。合伙协议约定：新合伙人按确定金额和财产份额入伙；合伙人退休或退伙时按财产份额公允价值退还；营运资金均来自合伙人；合伙人可按财产份额分得利润（利润分配自主决定）；清算时合伙人可按财产份额获得净资产。

**解析**：

① 退休或退伙时有交付金融资产的义务 → 满足金融负债定义；
② 但同时满足可回售工具分类为权益的5个条件：
   - 清算时按财产份额获得净资产 ✓
   - 合伙协议属于最次级类别（没有更次级的了）✓
   - 所有合伙人权益特征相同 ✓
   - 仅有回购义务，无其他现金交付义务 ✓
   - 现金流量基于企业损益、净资产变动 ✓

所以该金融工具应当确认为权益工具。

### 例题3：例13-20——合并报表中的处理

**题目**：甲公司控制乙公司。乙公司资本结构中有一部分是可回售工具（甲公司持有一部分，外部投资者持有一部分），这些可回售工具在乙公司个别报表中符合权益工具分类。

**解析**：

① 甲公司持有的乙公司可回售工具：在甲公司个别报表中按投资处理；合并时抵销。
② 外部投资者持有的乙公司可回售工具：在乙公司个别报表中是权益；在甲公司合并报表中，这部分对应的少数股东权益**不能列示为少数股东权益**，应当列示为**金融负债**。

**分录思路**（合并调整）：

借：少数股东权益（乙公司个别报表中的可回售工具对应部分）
贷：金融负债

含义：把子公司个别报表中"权益化"的可回售工具，在合并层面"还原"为负债，避免集团层面高估权益。

### 表格1：可回售工具分类为权益的5个条件

| 条件序号 | 具体要求 | 白话解释 |
|---------|---------|---------|
| 1 | 清算时按比例份额获得净资产 | 散了伙按持股比例分家产 |
| 2 | 该类别次于其他所有工具类别 | 排队分钱的排在最后面 |
| 3 | 该类别所有工具特征相同 | 同一类别的兄弟长得都一样 |
| 4 | 除回购/赎回义务外无其他现金义务 | 只有"退股"时要付钱，平时不分红不付息 |
| 5 | 现金流基于损益、净资产变动、公允价值变动 | 赚多少分多少，不保本不保息 |

### 表格2：可回售工具 vs 清算时交付净资产工具的特征对比

| 特征要求 | 可回售工具 | 清算时交付净资产工具 |
|---------|-----------|-------------------|
| 清算时按比例获得净资产 | ✓ 必须 | ✓ 必须 |
| 最次级类别 | ✓ 必须 | ✓ 必须 |
| 类别内所有工具特征相同 | ✓ 必须 | 仅要求清算时特征相同 |
| 无其他现金交付义务 | ✓ 必须 | 不要求（清算才是唯一条件） |
| 存续期现金流基于损益/净资产 | ✓ 必须 | 不要求 |
| 适用场景 | 开放式基金、合伙企业份额 | 封闭式基金、信托计划、有限寿命主体 |

### 表格3：特殊金融工具在个别报表 vs 合并报表中的分类

| 报表层面 | 发行方个别报表 | 母公司合并报表（少数股东部分） | 持有方报表 |
|---------|--------------|-------------------------|-----------|
| 分类 | 权益工具（满足条件时） | 金融负债 | 不属于权益工具投资 |
| 原因 | 最次级+清算按比例分净资产 | 例外不允许扩大到合并层面少数股东权益 | 对发行方是破例，对投资方不是 |
| 能否指定为其他权益工具投资 | 不适用 | 不适用 | 不能 |

### 例题4：含有额外分红义务的可回售工具

**题目**：某企业发行可回售工具，合同约定持有方有权按约定价格将工具回售给企业，同时企业每年须按净利润的10%向持有方分配利润。

**解析**：虽然该工具有可回售特征，但合同同时约定了每年按净利润一定比例分配的条款。这构成了一项交付现金的合同义务（分红义务），不满足"除了回购/赎回义务外无其他现金交付义务"的条件。因此，该可回售工具**不能适用特殊例外**，应当分类为金融负债。

**对比分录**：

| 情形 | 发行方分录 |
|-----|-----------|
| 符合特殊条件的可回售工具 | 借：银行存款 贷：其他权益工具 |
| 含额外分红义务的可回售工具 | 借：银行存款 贷：应付债券/其他金融负债 |

含义：同样是收到钱，能不能叫"权益"，关键看除了退股义务外还有没有其他必须付钱的承诺。

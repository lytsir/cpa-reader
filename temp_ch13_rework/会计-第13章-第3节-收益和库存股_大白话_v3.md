# （九）收益和库存股

## 核心提炼

这一段讲的是：金融工具分完类（负债还是权益）之后，后续的利息、股利、利得损失该怎么记账，以及回购自家股票（库存股）和每股收益计算要注意什么。一句话——**是负债就记损益，是权益就记权益变动**。

打个比方：你开了一家店，跟朋友借了10万元，每年付利息——这利息是经营成本，要从利润里扣。但如果你跟朋友约定这10万元算他入股，每年分红——这分红是从利润里分出去的，不影响利润表的"净利润"数字。金融工具分类不同，利息和分红的记账位置完全不同。

## 卡点引导

读到这里，你可能会问几个问题：

- **卡点1：同样是发永续债，利息有的计入当期损益，有的计入利润分配，区别在哪？**
- **卡点2：复合金融工具（比如可赎回优先股）的利息和股利怎么拆开来记账？**
- **卡点3：交易费用为什么有的冲权益、有的进损益？怎么区分？**
- **卡点4：库存股为什么不能确认金融资产？回购再出售为什么不产生损益？**
- **卡点5：每股收益计算时，其他权益工具的股利为什么要从净利润里扣掉？**

## 逐一破解

### 卡点1：同样是发永续债，利息有的计入当期损益，有的计入利润分配，区别在哪？

区别就在分类：

- **分类为金融负债**的永续债：利息支出计入当期损益（利润表里的财务费用），跟银行借款利息一样处理。
- **分类为权益工具**的永续债：利息支出计入利润分配（比如"应付股利"），跟给股东分红一样处理。

为什么？因为负债是你欠别人的钱，付利息是在"还钱的成本"；权益是别人投给你的钱，付"利息"实际上是在分利润。名头虽然都叫"利息"，但经济实质不同。

| 分类 | 利息/股利处理 | 公允价值变动 | 发行/回购/再融资损益 |
|------|--------------|-------------|---------------------|
| 金融负债 | 计入当期损益 | 计入当期损益 | 计入当期损益 |
| 权益工具 | 计入利润分配 | 不确认 | 作为权益变动处理 |

### 卡点2：复合金融工具的利息和股利怎么拆开记账？

复合金融工具既有负债成分，又有权益成分。记账原则是"谁的成分，谁的待遇"：

- **负债成分**相关的利息、利得损失 → 计入当期损益
- **权益成分**相关的股利、利得损失 → 计入权益

举个例子：企业发行5年后强制赎回的非累积优先股。赎回金额的折现值是负债成分，用实际利率法确认的利息支出进损益；跟权益成分相关的股利支付进利润分配。

但如果赎回不是强制的，而是看持有方要不要赎回——那上述处理仍然适用。不过，如果赎回金额还包括没支付的股利，整个工具就变成负债了，这时候所有股利都得进损益。

### 卡点3：交易费用为什么有的冲权益、有的进损益？

交易费用怎么分，关键看这笔交易是什么性质：

- **与权益交易相关**的费用 → 从权益（资本公积）中扣减，资本公积不够就依次冲盈余公积、未分配利润
- **不是权益交易**的费用 → 计入当期损益

实务中容易迷糊的是IPO：IPO时既发新股，又把老股上市流通。这时候要判断哪些费用跟"发新股"相关（冲权益），哪些跟"让老股上市"相关（进损益）。跟多项交易相关的共同费用，要在合理基础上分摊。

| 交易类型 | 交易费用处理 |
|---------|-------------|
| 发行新股 | 冲减资本公积（权益） |
| 购买库存股 | 减少所有者权益 |
| 将已发行股份上市流通 | 计入当期损益 |
| 发行复合金融工具 | 按负债/权益成分占总发行价的比例分摊 |

### 卡点4：库存股为什么不能确认金融资产？回购再出售为什么不产生损益？

库存股是企业回购自己的股票。准则明确规定：

- 回购支付的对价和交易费用，**减少所有者权益**，不得确认金融资产。
- 集团合并范围内其他成员（比如子公司）回购母公司股票，也算库存股；但联营企业和合营企业回购的，不算。
- 金融机构代客户持有自家股票，也不是库存股，因为那是替别人拿的，不是自己的资产。

为什么回购再出售不产生损益？因为你把库存股卖掉，本质上是**股东之间的股权转让**——你把钱给一个股东，把股票给另一个股东，公司本身没赚没亏。所以支付或收取的对价直接在权益里确认，不通过利润表。

| 事项 | 会计处理 | 是否影响损益 |
|------|---------|-------------|
| 回购自身股票 | 减少所有者权益 | 否 |
| 持有库存股期间公允价值变动 | 不确认 | 否 |
| 重新出售库存股 | 对价在权益中确认 | 否 |
| 子公司回购母公司股票 | 视同库存股处理 | 否 |

### 卡点5：每股收益计算时，其他权益工具的股利为什么要从净利润里扣掉？

每股收益（EPS）算的是"普通股股东"每股赚了多少钱。如果公司还发行了优先股等其他权益工具，这些工具的持有者也会分走一部分利润。算普通股EPS时，要把不属于普通股股东的部分扣掉：

**基本每股收益**：

| 项目 | 处理规则 |
|------|---------|
| 分子（归属于普通股股东的净利润） | 扣除其他权益工具的股利或利息 |
| 不可累积优先股 | 扣除**当期已宣告发放**的股利 |
| 累积优先股 | 无论当期是否宣告，**均应扣除**（因为累积的迟早要付） |
| 参加剩余利润分配的其他权益工具 | 扣除按可参加机制计算的应归属部分 |
| 分母 | 发行在外普通股的加权平均股数 |

**稀释每股收益**：如果金融工具含转股条款（比如可转优先股、可转债），计算时要考虑潜在稀释性，处理原则跟可转债、认股权证一样。

## 例题详解

### 【例题】权益工具永续债的利息处理

**案例事实**：甲公司发行永续债，分类为权益工具，每年按合同支付利息。

**四问**：

| 问题 | 解答 |
|------|------|
| 永续债分类为什么？ | 权益工具 |
| 利息支出计入哪里？ | 利润分配（如"应付股利"），不是财务费用 |
| 发行时的交易费用怎么处理？ | 从资本公积中扣减，不够冲减的依次冲盈余公积、未分配利润 |
| 如果以后赎回该永续债？ | 作为权益变动处理，赎回对价与账面价值的差额调整权益，不影响损益 |

**分录**：

| 事项 | 分录 | 借方含义 | 贷方含义 |
|------|------|----------|----------|
| 宣告发放永续债利息 | 借：利润分配——应付股利<br>贷：应付股利 | 利润减少（因为实质是分红） | 确认应付义务 |
| 实际支付利息 | 借：应付股利<br>贷：银行存款 | 冲减应付义务 | 现金流出 |
| 发行时支付交易费用 | 借：资本公积——股本溢价<br>贷：银行存款 | 冲减权益 | 现金流出 |

### 【例题】复合金融工具（可赎回优先股）的处理

**案例事实**：企业发行5年后强制赎回的非累积优先股，存续期内可自行决定是否支付股利。赎回金额为固定金额。

**四问**：

| 问题 | 解答 |
|------|------|
| 该工具是什么类型？ | 复合金融工具——负债成分（赎回金额折现值）+ 权益成分（剩余部分） |
| 负债成分的利息怎么处理？ | 按实际利率法确认利息支出，计入当期损益 |
| 权益成分的股利怎么处理？ | 实际支付时确认为利润分配 |
| 发行交易费用怎么分摊？ | 按负债成分和权益成分各自占总发行价款的比例分摊 |

**分录**：

| 事项 | 分录 | 借方含义 | 贷方含义 |
|------|------|----------|----------|
| 发行时确认负债成分 | 借：银行存款<br>贷：应付债券（负债成分） | 收到现金 | 确认负债 |
| 发行时确认权益成分 | 借：银行存款<br>贷：其他权益工具（权益成分） | 收到现金 | 确认权益 |
| 确认负债成分利息（实际利率法） | 借：财务费用——利息支出<br>贷：应付利息/应付债券——利息调整 | 损益增加（费用） | 负债增加 |
| 支付权益成分股利 | 借：利润分配——应付优先股股利<br>贷：应付股利 | 利润减少 | 确认应付股利 |

**变式**：如果赎回金额还包括"未支付的股利"，则整个工具都是负债，所有股利都进损益。

| 情形 | 分类 | 股利处理 |
|------|------|---------|
| 赎回金额=固定金额 | 复合工具（负债+权益） | 负债成分利息进损益，权益成分股利进利润分配 |
| 赎回金额=固定金额+未付股利 | 金融负债（整体） | 所有股利都进损益 |

### 【例题】库存股的处理

**案例事实**：甲公司以1000万元回购自身股票作为库存股，6个月后以1200万元出售。

**四问**：

| 问题 | 解答 |
|------|------|
| 回购时是否确认金融资产？ | 否，库存股不是资产，直接减少所有者权益 |
| 回购时交易费用怎么处理？ | 一并减少所有者权益 |
| 持有期间公允价值涨到1500万元，是否确认收益？ | 否，库存股不确认公允价值变动 |
| 出售时售价1200万元，是否确认200万元收益？ | 否，收到的对价直接在权益中确认，不走损益 |

**分录**：

| 事项 | 分录 | 借方含义 | 贷方含义 |
|------|------|----------|----------|
| 回购股票 | 借：库存股 10000000<br>贷：银行存款 10000000 | 权益减少（库存股是所有者权益的抵减项） | 现金流出 |
| 出售库存股 | 借：银行存款 12000000<br>贷：库存股 10000000<br>贷：资本公积——股本溢价 2000000 | 收到现金 | 冲减库存股成本；差额增加资本公积（权益内部结构调整） |

**注意**：如果售价低于回购成本，差额依次冲减资本公积、盈余公积、未分配利润。

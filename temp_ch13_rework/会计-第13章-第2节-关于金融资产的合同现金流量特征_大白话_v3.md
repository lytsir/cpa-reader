锚点ID: 会计-第13章-第2节-关于金融资产的合同现金流量特征

---

## 一、核心提炼

这一节讲的是：想把金融资产归类为"摊余成本计量"或"其他综合收益"，必须先看它的合同现金流是不是"老老实实还本付息"。如果是股票、基金这种收益不确定的，对不起，只能进"交易性金融资产"。

换句话说，准则设置了一道门槛——合同现金流量特征必须满足"本金加利息"（简称SPPI测试）。只有现金流稳定、可预测、像银行贷款那样的基本借贷安排，才有资格享受摊余成本这种"安稳"的计量方式。

---

## 二、卡点引导

**你可能会问**：什么是"本金加利息"？跟平时说的本金利息一样吗？

**这里容易晕**：利息里面到底包了哪些东西？时间价值、信用风险、管理费都算吗？

**你可能会问**：合同里写了可以提前还款，是不是就不符合要求了？

**这里容易晕**：银行把贷款基准利率换成LPR报价利率，这会影响分类吗？

**你可能会问**：有抵押担保的资产，合同现金流量评估会不会更宽松？

---

## 三、逐一破解

### 卡点1：什么是"本金加利息"？

"本金加利息"不是口语里的"借了100万，还110万"这么简单。准则里的定义很严格：

- **本金**：是金融资产在初始确认时的公允价值。注意，它不是一成不变的——如果合同允许提前还款，本金金额在存续期内会发生变动。
- **利息**：是对四样东西的对价——①货币时间价值；②信用风险；③其他基本借贷风险（如流动性风险）；④成本和利润。

**比喻**：就像你把100万存进银行定期。银行到期还你100万本金，再给你利息。这利息里，一部分是因为"钱放久了该多给"（时间价值），一部分是因为"银行可能赖账"（信用风险），还有一部分是银行的管理费和利润。这些加在一起，都算"符合要求的利息"。

但如果银行说："利息跟股市涨跌挂钩"，那就不行了。因为股市风险不是基本借贷风险，收益超出了"本金加利息"的范畴。

### 卡点2：利息里包了哪些东西？

利息的组成，准则明确列出了"许可范围"：

| 项目 | 是否属于利息 | 说明 |
|------|------------|------|
| 货币时间价值 | 是 | 仅因时间流逝而提供的对价，是利息的核心 |
| 信用风险 | 是 | 与未偿付本金相关的违约风险补偿 |
| 流动性风险 | 是 | 资金被锁定无法随时取用的补偿 |
| 管理费用 | 是 | 出借方的基本运营成本 |
| 利润 | 是 | 出借方的合理回报 |
| 权益风险 | 否 | 与股价、股指挂钩的不属于 |
| 商品价格风险 | 否 | 与黄金、原油价格挂钩的不属于 |

**关键红线**：货币时间价值要素有时可能存在"修正"，比如计息方式不是标准的。这时企业要评估修正后的现金流是否还满足要求。如果修正幅度很小、不影响"基本借贷"的本质，那就没问题。

### 卡点3：合同有特殊条款怎么办？

很多贷款或债券合同里写了"可以提前还款"或"可以展期"。这种条款会不会让资产"出局"？

准则说了：**要看影响大不大**。

- 如果提前还款权的公允价值非常小（比如几乎没人会提前还，或者提前还的罚息很高），那这个条款不影响整体判断。
- 如果展期条款只是给借款人一个选择权，且展期后的现金流仍然是本金加利息，那也可以接受。

**比喻**：就像租房合同里写了"租客可提前退租，但需支付2个月违约金"。因为违约金很高，实际上很少有人会提前退，所以这个条款不改变"租客按月交租"的本质。

### 卡点4：LPR改革对分类有影响吗？

实务中有个具体问题：以前贷款按"贷款基准利率"定价，后来央行改成LPR报价利率，这会否导致原本符合要求的资产变得不符合？

准则明确回答：**单纯从基准利率改为LPR，不影响分类**。

- 利率为"LPR + 200基点"的贷款——符合本金加利息特征。
- 利率为"LPR向上浮动20%"的贷款——**不符合**，因为包含杠杆因素。

**关键区别**：固定加点（+200基点）只是换了个定价基准，本质还是"本金加利差"。但"向上浮动20%"引入了杠杆——LPR涨1%，你的利率涨1.2%，收益波动被放大了，不再是单纯的借贷安排。

### 卡点5：抵押担保和追索权影响评估吗？

不影响。

准则说得很清楚：企业应当使用金融资产的**计价货币**来评估合同现金流量特征。完全追索权和抵押品担保这些增信措施，不改变合同现金流本身的性质。

**大白话**：抵押只是"还不上了有东西抵债"，不代表合同本身的现金流变成了股票分红或商品收益。该是什么还是什么。

## 四、例题详解

### 例题1：普通债券是否符合SPPI？

甲公司购入乙公司发行的普通公司债券，面值1000万元，票面年利率5%，每年付息一次，到期还本。无转股条款、无提前还款条款、无其他特殊安排。

**判断**：符合本金加利息的合同现金流量特征。

**理由**：现金流仅为到期收回本金1000万元，以及每年按5%收取的固定利息。利息仅包含时间价值、信用风险等基本借贷要素，无权益、商品或其他非借贷风险挂钩。

**会计分类**：若业务模式为收取合同现金流量，则分类为"债权投资"（以摊余成本计量）。

### 例题2：可转换债券是否符合SPPI？

甲公司购入丙公司可转换债券，面值500万元，票面年利率2%，可在第3年末按约定比例转为丙公司普通股。

**判断**：不符合本金加利息的合同现金流量特征。

**理由**：该债券嵌入了一项转股权。持有人除了收取固定利息外，还可能因转股而获得与股票价格挂钩的收益。这使得整体现金流包含了"基本借贷安排以外的其他因素变动的不确定性"。准则明确要求：可转换债券不单独分拆转股权，而是作为一个整体评估。

**会计分类**：只能分类为"交易性金融资产"（以公允价值计量且其变动计入当期损益）。

### 例题3：结构性存款是否符合SPPI？

甲公司在某商业银行购买结构性存款100万元，产品说明书约定：收益与黄金期货价格挂钩，若到期时黄金价格高于期初价格，则获得年化4%收益；若低于，则仅获得年化0.5%收益。

**判断**：不符合本金加利息的合同现金流量特征。

**理由**：该存款嵌入了与商品价格（黄金期货）挂钩的衍生条款。现金流量随黄金价格波动而变动，不属于仅为对本金和利息的支付。结构性存款本质上是"存款+衍生品"的组合。

**会计分类**：应当分类为"交易性金融资产"。

### 例题4：含提前还款权的贷款是否符合SPPI？

甲银行向乙企业发放贷款1000万元，年利率6%，期限3年。合同约定乙企业可在第2年末提前还款，提前还款无需支付额外费用，且提前还款金额等于当时摊余成本。

**判断**：需要进一步评估。

**分析步骤**：
1. 先判断提前还款权本身的公允价值是否非常小。
2. 如果该权利几乎不会被行使（例如市场利率高于6%，借款人没有提前还款动力），或行使后对现金流的影响微乎其微，则不影响SPPI判断。
3. 如果提前还款权公允价值较大，可能改变现金流的时间分布和金额，则需判断是否仍满足"本金加利息"特征。

**会计分类**：若评估后仍符合SPPI，且业务模式为收取合同现金流量，则分类为"贷款"（以摊余成本计量）。

## 五、表格总结

### 表1：利息的"许可范围"

| 构成要素 | 是否属于利息 | 举例 |
|---------|------------|------|
| 货币时间价值 | 是 | 资金占用的时间成本 |
| 信用风险溢价 | 是 | 对借款人可能违约的补偿 |
| 流动性风险 | 是 | 资金被长期锁定的补偿 |
| 管理费用 | 是 | 贷款审批、管理的成本 |
| 合理利润 | 是 | 出借方的经营回报 |
| 杠杆因素 | 否 | 利率按基准上浮百分比 |
| 权益/商品价格联动 | 否 | 与股市、黄金、汇率挂钩 |

### 表2：特殊条款的评估要点

| 特殊条款 | 评估方法 | 是否影响SPPI |
|---------|---------|------------|
| 提前还款权 | 评估其公允价值是否非常小 | 若非常小，不影响 |
| 展期权 | 展期后现金流是否仍为本金加利息 | 若是，不影响 |
| 利率调整（LPR改革） | 是否仅更换定价基准 | 若仅为基准替换，不影响 |
| 利率含杠杆 | 利率变动是否被放大 | 若含杠杆，不符合 |

### 表3：常见金融资产的SPPI判断

| 资产类型 | 是否符合SPPI | 会计分类 |
|---------|------------|---------|
| 普通债券（固定/浮动利率） | 符合 | 债权投资或其他债权投资 |
| 股票 | 不符合（股利不是利息） | 交易性金融资产/其他权益工具投资 |
| 开放式基金 | 不符合（含处置现金流） | 交易性金融资产 |
| 可转换债券 | 不符合（含转股权） | 交易性金融资产 |
| 结构性存款 | 不符合（与商品/权益挂钩） | 交易性金融资产 |
| 应收账款（正常信用期） | 符合 | 应收账款（以摊余成本计量） |

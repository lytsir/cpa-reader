# 继续涉入被转移金融资产的会计处理

## 核心提炼

企业既没有转移也没有保留金融资产所有权上几乎所有的风险和报酬，但保留了对该金融资产控制的，需要按继续涉入的程度继续确认该资产，并确认相关负债。核心逻辑是"你保留了多大权利和义务，就确认多少资产和负债"。资产和负债的计量基础取决于原金融资产的计量方式（摊余成本或公允价值）。最常见的继续涉入方式是提供担保。

## 卡点引导

这一节的难点通常在于：

- 什么情况下会进入"继续涉入"的会计处理？
- 继续涉入资产和继续涉入负债的金额怎么确定？
- 提供担保方式继续涉入时，"账面价值和担保金额孰低"是什么意思？
- 继续涉入仅限于金融资产一部分时，账面价值怎么在继续确认部分和不再确认部分之间分配？
- 投资收益110万元是怎么算出来的？

## 逐一破解

**适用继续涉入会计处理的前提条件**

必须同时满足：
- 既没有转移也没有保留所有权上几乎所有的风险和报酬
- 保留了对该金融资产的控制

此时，企业应当按照继续涉入被转移金融资产的程度，继续确认该被转移金融资产，并相应确认相关负债。所确认的资产和负债应当反映企业所保留的权利和承担的义务。

**继续涉入资产和负债的后续处理**

- 对继续涉入形成的有关资产确认相关收益
- 对继续涉入形成的相关负债确认相关费用
- 继续涉入资产应根据原金融资产的性质和分类，继续列报于资产负债表中的贷款、应收款项等项目

**计量基础的选择**

被转移金融资产以摊余成本计量的：被转移资产和相关负债的账面价值等于企业保留的权利和义务的摊余成本。

被转移金融资产以公允价值计量的：被转移资产和相关负债的账面价值等于企业保留的权利和义务按独立基础计量的公允价值。

重要限制：如果所转移的金融资产以摊余成本计量，相关负债不得指定为以公允价值计量且其变动计入当期损益的金融负债。

**提供担保方式的继续涉入**

最常见的情形。转移日按以下规则确认：

继续涉入资产 = 金融资产的账面价值与担保金额两者之中的较低者

相关负债 = 担保金额 + 担保合同的公允价值

担保金额指企业所收到的对价中，将可能被要求偿还的最高金额。担保合同的公允价值通常指提供担保而收取的费用。

**仅限于金融资产一部分的继续涉入**

按照转移日继续确认部分和不再确认部分的相对公允价值，在两者之间分配金融资产的账面价值。

不再确认部分的损益 = 分配至不再确认部分的账面金额（转移日） - 不再确认部分所收到的对价

如果涉及转移的金融资产分类为以公允价值计量且其变动计入其他综合收益，不再确认部分对应的原计入其他综合收益的公允价值变动累计额应当计入当期损益。

## 例题详解

**例题：提供担保方式的继续涉入**

甲银行与乙银行签订贷款转让协议，将本金1000万元、年利率10%、期限9年的组合贷款出售给乙银行，售价990万元。双方约定，甲银行为该笔贷款提供担保，担保金额为300万元，实际贷款损失超过担保金额的部分由乙银行承担。转移日该笔贷款（包括担保）的公允价值为1000万元，其中担保的公允价值为100万元。甲银行没有保留对该笔贷款的管理服务权。该贷款没有市场，乙银行不具备出售该笔贷款的实际能力。

分析步骤：

第一步，判断是否适用继续涉入会计处理。

- 甲银行出售贷款但提供担保，既没有转移也没有保留所有权上几乎所有的风险和报酬（担保意味着仍承担部分风险）
- 该贷款没有市场，乙银行不具备出售该笔贷款的实际能力，说明甲银行保留了对该贷款的控制
- 因此应当按照继续涉入的程度继续确认该被转移金融资产

第二步，确定继续涉入资产金额。

金融资产的账面价值为1000万元，担保金额为300万元，按较低者确认继续涉入资产，即300万元。

第三步，确定相关负债金额。

相关负债 = 担保金额300万元 + 担保合同公允价值100万元 = 400万元。

第四步，计算投资收益。

借方合计 = 存放中央银行款项990万元 + 继续涉入资产300万元 = 1290万元
贷方合计 = 贷款1000万元 + 继续涉入负债400万元 = 1400万元

差额 = 1290 - 1400 = -110万元，计入投资收益借方，即投资损失110万元。

也可以用对价思路理解：收到的对价是990万元，终止确认贷款账面价值1000万元，同时保留了300万元的继续涉入资产和400万元的继续涉入负债，净影响 = 990 - (1000 - 300) - 400 = -110万元。

账务处理：

借：存放中央银行款项 9900000
借：继续涉入资产 3000000
借：投资收益 1100000
贷：贷款 10000000
贷：继续涉入负债 4000000

**例题：继续涉入仅限于一部分的分配问题**

接上例，假设甲银行仅对贷款组合中的500万元本金提供担保，其余500万元无担保出售。转移日该500万元无担保部分的公允价值为495万元，有担保部分的公允价值为505万元（其中担保公允价值仍为100万元）。

分析：贷款整体账面价值1000万元，需要按继续确认部分和不再确认部分的相对公允价值进行分配。

不再确认部分（无担保出售部分）的公允价值 = 495万元
继续确认部分（有担保部分）的公允价值 = 505万元

分配比例：不再确认部分 = 495 / (495 + 505) = 49.5%
继续确认部分 = 505 / 1000 = 50.5%

分配账面价值：
不再确认部分账面价值 = 1000 x 49.5% = 495万元
继续确认部分账面价值 = 1000 x 50.5% = 505万元

无担保部分售价假设为490万元（为与例题一致，此处沿用对价思路）：

不再确认部分损益 = 495 - 490 = 5万元（损失）

有担保部分：担保金额为300万元，但分配到的账面价值为505万元，继续涉入资产按较低者取300万元。相关负债 = 300 + 100 = 400万元。

投资收益总额需要综合两部分计算。

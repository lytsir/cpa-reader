## 一、为什么要有或有结算条款的规定

企业在发行金融工具（比如优先股、永续债）时，合同里经常会写一些"如果发生了某某事，我就得付钱"的条款。这些条款的触发条件是未来的不确定事项——可能是股价涨跌、利率变化、国家政策调整，也可能是企业自身的财务状况变化。

准则制定者发现一个漏洞：如果允许企业随便设置这些"或有条款"，企业就可能钻空子——表面上说"这工具是股不是债"，实际上合同里埋了各种触发条件，一旦触发就得还本付息。这样会导致企业负债被严重低估，投资人也被忽悠了。

所以准则定了一条铁律：**只要发行方不能无条件地避免付钱（还本或付息），这个金融工具就必须分类为金融负债，不能算权益。** 但有3个例外情况，满足任一即可分类为权益工具。

---

## 二、或有结算条款是什么

### （一）一句话说清楚

或有结算条款，就是合同里写的"如果未来发生了某件不确定的事，结算方式就要变"的条款。这件不确定的事，必须是**发行方和持有方都控制不了**的。如果有一方能控制，那就不叫"或有"了。

这些不确定事项包括：
- 外部市场类的：股价指数变动、消费价格指数（CPI）变动、利率变动、税法变动
- 企业自身类的：未来收入、净利润、资产负债率等财务指标

### （二）核心判断标准：能不能"赖掉"

判断一个带有或有结算条款的金融工具是债还是股，核心就看一句话：

> **发行方能不能无条件地避免交付现金、其他金融资产，或者以其他导致该工具变成金融负债的方式结算？**

- **能赖掉** → 不用付钱 → 分类为**权益工具**
- **赖不掉** → 必须付钱 → 分类为**金融负债**

### （三）三个例外（满足任一就算权益）

虽然原则上赖不掉就得算负债，但有以下3种情况可以"网开一面"，仍然分类为权益：

| 例外情况 | 通俗解释 |
|---------|---------|
| ① 触发条件几乎不可能发生 | 就像"如果太阳从西边出来"这种极端罕见、显著异常、几乎不可能发生的事 |
| ② 只有清算时才触发 | 公司正常经营时不用管，只有公司破产清算时才需要付钱 |
| ③ 特殊金融工具中的可回售工具 | 前面讲过的那类特殊工具，本身就分类为权益 |

**重点记住**：实务中例外①"几乎不可能发生"这个门槛非常高，不是随便说说的。很多你以为"不太可能"的事，在准则眼里都不算"几乎不可能"。

## 三、实务中的常见陷阱

### （一）永续债里埋了一堆触发条件

实务中，企业发行永续债时，合同里可能写了满满一页"一旦发生以下任一事项，永续债立即到期，必须还本付息"：

- 公司无力偿债、拖欠到期款项
- 停止或暂停支付大部分债务
- 持续经营能力出现重大不利变化
- 发生超过净资产10%的重大损失
- 收入、利润、资产负债率等财务指标承诺未达标
- 会计政策、税收政策变动导致财务状况重大变化
- 受到超过一定金额的处罚或被监管部门调查
- 高管出现重大诚信问题
- 实际控制人严重违约
- 控制权变更或信用评级被降级
- IPO失败
- 股票停牌超过一定期限

**大白话判断**：上面这些事项，虽然谁也不想发生，但哪一条都**不属于"极端罕见、显著异常且几乎不可能发生"**的情况。企业经营中完全有可能碰到。所以，甲公司**无法无条件避免**以现金偿还永续债的义务 → **必须分类为金融负债**。

### （二）利息只在有可供分配利润时才支付

还有一种永续债，合同条款说"每年付利息，但只有公司有可供分配利润时才付，没利润就可以不付"。

你可能觉得：没利润就可以不付，那不是能赖掉吗？

**错！** 准则的逻辑是：公司虽然现在没利润，但未来**有可能**有利润。一旦有了利润，公司就得付利息。公司不能"无条件"地避免这个义务——它只是"有条件"地推迟了，但这个条件未来完全可能满足。所以，甲公司仍然**不能无条件避免支付现金** → **必须分类为金融负债**。

## 四、例题精讲

### 【例题1】控股股东变更触发赎回

**【条件翻译】**

甲公司要发行优先股。条款规定：
- 平时：甲公司可以自己决定是否发股利（发不发我说了算）
- 特殊情况：如果**控股股东发生变更**（假设这件事甲公司控制不了），甲公司**必须按面值赎回**该优先股

**【思路拆解】**

第一步：找或有结算条款。"控股股东变更"是甲公司控制不了的未来不确定事项 → 属于或有结算条款。

第二步：判断这件事是否"几乎不可能发生"。控股股东变更在资本市场上虽然不天天发生，但也绝非"极端罕见、显著异常且几乎不可能发生"的事。

第三步：判断能否无条件避免。一旦控股股东变更，甲公司**必须**赎回 → 不能无条件避免交付现金的义务。

**【结论】**

该工具应当划分为**金融负债**。

**【陷阱提示】**

"自己能决定发不发股利"这个条款容易让人误以为这是权益工具。但别忘了判断的核心是"最不利情况"——一旦有触发事件，发行方能不能赖掉。如果有一个触发条件会让你赖不掉，那整个工具就是负债。

### 【例题2】IPO失败触发回购（补充协议：终止回售权）

**【条件翻译】**

背景信息：
- 甲公司是境内注册的股份制企业，控股股东是乙公司，拟在境内上市
- 2×21年1月1日，丙公司作为战略投资人向甲公司增资3亿元，持有20%股权
- 增资协议约定：如果甲公司**未能在2×24年12月31日前完成IPO**，丙公司有权要求甲公司以现金回购其股权
- 回购价格 = 增资3亿元 + 按8%年化收益率及实际投资期限计算的收益
- 这个回售权只有丙公司享有，不能随股权转让
- 甲公司回购股份需要履行减资程序

后来（2×21年6月30日），为推进上市进程，三方签补充协议：
- 自补充协议签署之日起，**终止**丙公司的回售权
- 但如果甲公司未能在2×24年12月31日前完成IPO，丙公司自2×25年1月1日起有权要求**乙公司**以现金购买其股权，但无权向甲公司提出回购

**【思路拆解】**

**时间点一：2×21年1月1日（签增资协议时）**

第一步：找或有结算条款。"能否在2×24年12月31日前完成IPO"是甲公司和丙公司都控制不了的未来不确定事项 → 属于或有结算条款。

第二步：判断是否"几乎不可能"。IPO失败虽然不期望发生，但在实务中绝非极端罕见 → 不属于"几乎不具有可能性"。

第三步：判断能否无条件避免。如果IPO失败，丙公司有权要求甲公司现金回购 → 甲公司**不能无条件避免**以现金回购自身权益工具的合同义务。

有人可能说："甲公司回购需要履行减资程序，这不是有限制吗？" 

**注意**："存在回购义务"和"履行回购义务的能力限制"是两码事。减资程序只是履行义务的法律步骤，不是解除了义务本身。就像你欠了钱，法律规定要签个字才能还，但不代表你可以不还。

**结论**：甲公司应当将收到的增资款先确认股本和资本公积（股本溢价），同时按回购所需支付金额的现值，将回购义务**从权益重分类为金融负债**。

**时间点二：2×21年6月30日（签补充协议后）**

补充协议的核心变化：甲公司的回购义务**终止**了——丙公司以后只能找乙公司（控股股东），不能找甲公司了。

这意味着：甲公司现在可以**无条件地避免**以现金回购自身权益工具的合同义务了。

**会计处理**：
- 终止确认之前确认的金融负债
- 确认一项新的权益工具
- 按该权益工具当日的公允价值计量
- **不可追溯调整**以前年度对丙公司增资的分类

**【数字推导】**

回购金额的现值计算：
- 本金：3亿元
- 年化收益率：8%
- 假设实际投资期限为T年
- 回购金额 = 30000万元 × (1 + 8% × T)
- 现值 = 回购金额按适当折现率折现到2×21年1月1日的金额

具体数字题目没给折现率，考试时会提供。关键是理解"按回购所需支付金额的现值"确认金融负债。

补充协议后，新权益工具的公允价值与终止确认金融负债账面价值的差额，**计入权益**（不是损益）。为什么？因为签署补充协议是为了符合法律和监管规定（清理特殊权益以便IPO），丙公司愿意接受是因为其股东身份和IPO后的利益。这种交易属于**权益性交易**，差额进权益。

**【陷阱提示】**

1. 减资程序≠可以免责。很多考生误以为"回购需要减资程序"意味着发行方可以避免义务，这是大坑。
2. 追溯调整陷阱。补充协议签订后不能追溯调整以前年度，只能按当日公允价值确认新权益工具。
3. 差额进权益不进损益。这是权益性交易，差额不能计入当期损益。

### 【例题3】IPO失败触发回购（补充协议：中止+恢复回售权）

**【条件翻译】**

承接例题2的背景，但补充协议条款变了：
- 2×21年6月30日签补充协议，约定自签署之日起**中止**丙公司的回售权
- 但如果甲公司未能在2×24年12月31日前完成IPO，则于2×25年1月1日**恢复**该回售权

**【思路拆解】**

表面上看，回售权被"中止"了，但补充协议同时埋了一个恢复条件：如果IPO失败，回售权就恢复。

这跟原来的增资协议有什么本质区别吗？

- 原协议：IPO失败 → 丙公司有权要求甲公司回购
- 补充协议：IPO失败 → 回售权恢复 → 丙公司有权要求甲公司回购

**完全没有实质差别！**

甲公司是否承担以现金回购自身权益工具的义务，仍然取决于"能否在2×24年12月31日前完成IPO"这个双方都控制不了的未来不确定事项。这仍然属于或有结算条款，且不属于"几乎不具有可能性"的情形。

**【结论】**

在2×21年6月30日，甲公司**不能无条件避免**以现金回购自身权益工具的合同义务 → **应当继续按2×21年1月1日的分类进行会计处理**，即保持金融负债的分类，不做调整。

如果甲公司后来在2×24年12月31日前成功完成IPO，丙公司丧失回售权，甲公司应当在上市日将丙公司的增资**重分类为权益工具**，初始确认金额按当日金融负债的账面价值计量。

**【陷阱提示】**

"中止"不等于"终止"。中文里差一个字，会计处理天差地别：

| | 例题2（终止） | 例题3（中止+恢复） |
|---|---|---|
| 补充协议签署日 | 金融负债→权益工具 | 继续确认金融负债 |
| 核心差异 | 回购义务彻底消灭 | 回购义务暂时休眠，条件触发即恢复 |
| 是否满足"无条件避免" | 是 | 否 |

考试时就考这一字之差！

## 五、总结记忆口诀

> 或有结算看控制，双方控制不了才算数。
> 能不能赖是核心，赖不掉就是金融负债。
> 三个例外要记牢：几乎不可能、清算才触发、特殊可回售。
> "几乎不可能"门槛高，别拿"不太可能"来充数。
> 减资程序不免责，存在义务和履行能力是两码事。
> 终止和中止差一字，会计处理天壤之别。

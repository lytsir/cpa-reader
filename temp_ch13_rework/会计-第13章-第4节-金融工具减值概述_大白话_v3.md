# 金融工具减值概述

## 核心提炼

金融工具减值从过去的"实际已发生损失法"变成了"预期信用损失法"。核心变化是：计提减值准备不再等违约实际发生了才动手，而是要在资产负债表日，就根据未来可能违约事件造成的损失的期望值来提前确认损失准备。简单说，以前是看"已经损失了多少"，现在是看"预计未来可能损失多少"。

## 卡点引导

理解预期信用损失法，最大的卡点是思维转变。以前学的是"有确凿证据表明发生减值了，才计提减值准备"，比如对方破产了、跑路了，才能计提。现在变成"只要有违约的可能，哪怕还没发生，也要按概率加权算一个期望值来计提"。

第二个卡点是"信用损失"的定义。不是简单算"对方欠多少没还"，而是要把所有合同应收的现金流和预计能收到的现金流做差额，再按原实际利率折现到当前。这意味着：即使最终预计能全额收回，但收回时间比合同晚了，也会产生信用损失（因为货币有时间价值）。

第三个卡点是"经信用调整的实际利率"这个特殊概念。正常用原实际利率折现，但如果买的是已经信用减值的资产（比如对方已经违约的债券），折现率要用经信用调整的实际利率。为什么？因为初始买入价已经反映了信用风险，如果还用原来的高利率折现，会高估损失。

## 逐一破解

### 1. 预期信用损失法的适用范围

不是所有金融资产都要按这个法子计提减值。准则明确规定了四类：
- 分类为以摊余成本计量的金融资产（比如债权投资、应收账款）
- 分类为以公允价值计量且其变动计入其他综合收益的金融资产（比如其他债权投资）
- 租赁应收款
- 合同资产和部分贷款承诺、财务担保合同

注意两个例外不在这个范围里的：
- 以公允价值计量且其变动计入当期损益的金融资产（比如交易性金融资产），公允价值变动已经反映了市场风险，不单独提减值
- 指定为以公允价值计量且其变动计入其他综合收益的权益工具投资，也不提减值

### 2. 损失准备的定义

损失准备不是单一概念，包括三块：
- 以摊余成本计量的金融资产、租赁应收款和合同资产的预期信用损失计提的准备
- 以公允价值计量且其变动计入其他综合收益的金融资产的累计减值金额
- 贷款承诺和财务担保合同的预期信用损失计提的准备

这里特别注意第二类：其他债权投资的减值不减少账面价值（因为公允价值计量），而是把减值金额计入其他综合收益，同时在利润表中确认信用减值损失。所以它的"损失准备"体现在其他综合收益里的累计减值金额。

### 3. 预期信用损失的定义

预期信用损失 = 以发生违约的风险为权重的信用损失加权平均值。

分解一下：
- 先算"信用损失"：合同应收现金流 - 预计收取现金流的差额，按原实际利率折现
- 再考虑各种违约情景的概率
- 最后加权平均

举个例子：一笔100万元的贷款，1年后到期。有三种情景：
- 90%概率正常还款，收100万元
- 8%概率延迟3个月还款，收100万元（但晚3个月）
- 2%概率违约，只能收回60万元

假设原实际利率为5%。信用损失计算：
- 正常情景：现金流无差异，信用损失=0
- 延迟情景：收100万元但晚3个月，信用损失 = 100/(1+5%)^(0.25) - 100/(1+5%)^1 = 折现差异
- 违约情景：应收100万元，实收60万元，信用损失 = 100/(1+5%)^1 - 60/(1+5%)^1 = 40/1.05

预期信用损失 = 90%×0 + 8%×(延迟损失) + 2%×(违约损失)

### 4. 信用损失的定义

信用损失是"全部现金短缺的现值"。这个定义有几个关键点：

第一，"现值"意味着必须折现，折现率用原实际利率（或经信用调整的实际利率）。不能简单用名义金额相减。

第二，"全部现金短缺"包括本金和利息的短缺。如果合同规定收100万元本金+5万元利息，预计只能收回90万元本金+3万元利息，短缺12万元，但这12万元要折现。

第三，即使预计最终全额收回，只要收款时间晚于合同期限，折现后的差额就是信用损失。比如合同规定1年后收100万元，但预计1年半后才能收100万元，折现到1年时点，100/(1+r)^1.5 和 100/(1+r)^1 的差额就是信用损失。

第四，对于已发生信用减值的金融资产，折现率改用"经信用调整的实际利率"。这个利率在购买日就已经把初始信用损失考虑进去了，所以比原实际利率低。用这个低利率折现，算出来的信用损失更符合经济实质。

### 5. 估计现金流时的注意事项

估计未来现金流要考虑：
- 整个预计存续期的所有合同条款（提前还款、展期、看涨期权等）
- 出售担保品能收回的现金流
- 合同条款组成部分的其他信用增级（比如财务担保）

但要注意不重复计算：如果某个信用增级已经单独确认（比如单独作为一项资产或负债），在计量预期信用损失时就不能再考虑它产生的现金流。

对于组合层面的信用增级（比如给一篮子贷款买的信用保险，有共享赔付上限），如果不能可靠证明和单项金融工具的对应关系，单项金融工具计量预期信用损失时也不考虑。

## 例题详解

### 例题1：基本概念辨析

问：预期信用损失法和实际已发生损失法的根本区别是什么？

答：根本区别在于确认时点和前提不同。

实际已发生损失法：必须等"减值迹象"出现、"确凿证据"存在，才能确认损失。比如对方已经破产清算、连续违约90天以上。这是一种"事后"视角。

预期信用损失法：在资产负债表日，不管是否已经发生违约，都要基于未来可能违约的概率和损失金额，计算一个期望值来确认损失。这是一种"事前"视角。

举例：甲公司持有乙公司债券。乙公司目前经营正常，但行业整体下滑，未来2年违约概率从1%上升到5%。
- 实际已发生损失法：乙公司没违约，不计提减值
- 预期信用损失法：虽然还没违约，但违约概率上升了，要根据5%的违约概率和违约后的损失率计算预期信用损失，计提减值准备

### 例题2：信用损失计算

甲公司发放一笔贷款，面值1000万元，期限3年，年利率6%，每年末付息60万元，到期还本。甲公司计算的实际利率为6%。

假设在第一个资产负债表日，甲公司预计：
- 70%概率正常履约，每年收60万元利息，到期收1000万元本金
- 20%概率第2年末开始违约，第2年收30万元，第3年收500万元
- 10%概率第2年末完全违约，只能收回抵押品变现400万元

信用损失计算（折现率6%）：

正常情景（70%）：现金流无短缺，信用损失=0

部分违约情景（20%）：
- 合同应收：第1年60万，第2年60万，第3年1060万
- 预计实收：第1年60万，第2年30万，第3年500万
- 短缺：第2年30万，第3年560万
- 折现到当前：30/(1.06)^2 + 560/(1.06)^3 = 26.70 + 470.12 = 496.82万元
- 加权：20%×496.82 = 99.36万元

完全违约情景（10%）：
- 合同应收：第1年60万，第2年60万，第3年1060万
- 预计实收：第1年60万，第2年400万（抵押品变现），第3年0
- 注意：400万在第2年末收到
- 短缺：第2年少收60万（因为400>60，但本金还差），实际短缺 = 60/(1.06)^1 + 1060/(1.06)^3 - 多收的340万/(1.06)^2
- 更准确的算法：合同现金流现值 - 预计现金流现值
  = [60/1.06 + 60/1.06^2 + 1060/1.06^3] - [60/1.06 + 400/1.06^2 + 0]
  = 1000 - [56.60 + 356.00]
  = 1000 - 412.60
  = 587.40万元
- 加权：10%×587.40 = 58.74万元

预期信用损失 = 0 + 99.36 + 58.74 = 158.10万元

甲公司应在第一个资产负债表日计提158.10万元的损失准备。

### 例题3：延迟收款也产生信用损失

甲公司持有一笔应收账款100万元，合同规定6个月后到期。客户经营状况正常，预计能全额付款，但会延迟3个月（即9个月后才付款）。假设原实际利率为8%。

问：是否产生信用损失？

答：产生。虽然预计最终能全额收回100万元，但收款时间比合同晚了3个月。

信用损失 = 100/(1+8%)^0.75 - 100/(1+8%)^0.5
（按年折现：9个月是0.75年，6个月是0.5年）

= 100/1.059 - 100/1.039
= 94.43 - 96.25
= -1.82万元

等等，这个算法有问题。正确的理解是：合同应收款的现值（在当前时点） vs 预计实收款的现值（在当前时点）。

合同应收款现值（站在当前）：100/(1+8%)^0.5 = 96.23万元
预计实收款现值（站在当前）：100/(1+8%)^0.75 = 94.50万元

信用损失 = 96.23 - 94.50 = 1.73万元

这个例子说明：只要收款时间晚于合同规定，即使金额不少，也会产生信用损失。损失的大小取决于延迟时间和折现率。

【阶段二：具象化方案】

用"借钱给别人，实际回报率和合同写的不一样"来比喻实际利率。你花1000万买了一份面值1250万的债券，票面利率4.72%，每年收59万利息，但这不够弥补你少花的250万，所以实际回报率高于4.72%。摊余成本就是"这笔钱在你账上实际值多少钱"，每年按实际利率涨一点，涨到到期时刚好等于面值。三种情形分别对应：正常还本、中途改约定、到期一次拿钱。用"滚雪球"比喻到期一次还本付息的情形，利息不拿走，越滚越大。

【阶段三：大白话正文】

一、为什么要用实际利率法和摊余成本

企业买债券，花的钱和面值往往不一样。可能花1000万买了面值1250万的债券，也可能花1050万买了面值1000万的债券。如果只看票面利率，算出来的利息收入是假的，因为它没有反映你实际投入的资金成本。

举个例子，你花90元买了一张面值100元的债券，票面利率5%，每年收5元利息。如果只按5%算收益，你觉得自己赚了5元。但你实际只花了90元啊，真实的回报率肯定比5%高。实际利率法就是要把你真正投入的钱、真正收回的钱，按一个统一的回报率来计算每期的真实收益。

摊余成本就是记录这笔投资在账上"实际值多少钱"的方法。它从初始投入开始，每期加上实际应得收益，减去实际拿到手的钱，剩下的就是下一期的本金基数。到期时，摊余成本应该刚好等于面值（或最终收回金额）。

如果不这样做，企业就可以通过折价或溢价购买债券来操纵利润。比如折价买了债券，如果一次性把折扣全部确认为收益，就不符合配比原则。实际利率法把折扣或溢价分摊到各期，让每期的收益都反映真实的资金回报。

二、什么是实际利率和摊余成本

实际利率，就是把未来所有能拿到的钱（利息和本金），折现成你现在花出去的钱，所用的那个折现率。它考虑了所有合同条款，但不考虑信用损失。换句话说，它是这笔投资的真实年化回报率。

怎么理解折现？假设你1年后会收到110元，如果市场利率是10%，那这110元在今天值100元。实际利率就是反过来的过程：已知未来能收到多少钱，已知现在花了多少钱，求那个让两边相等的利率。

经信用调整的实际利率，是专门用于已经信用减值的金融资产。它在估计未来现金流时，把初始就发生的信用损失也考虑进去了。比如你买了一个已经违约的债券，花了500万，预计只能收回800万，那折现率就要反映这个现实。

摊余成本，是金融资产的"实际账面价值"。计算公式可以概括为：初始金额，减去已收回本金，加减累计摊销额（利息调整的摊销），再减去减值准备。简单记就是：你实际投进去的钱，加上该赚还没拿到手的收益，减去已经拿回来的本金。

对于浮动利率工具，如果初始金额等于到期本金，那利率波动通常不会对账面价值产生重大影响，因为利率涨了，利息多收了，折现的影响相互抵消。

三、具体怎么做

1. 计算实际利率

实际利率的计算思路是：让未来现金流的现值等于现在的投资成本。公式看起来复杂，其实就是一个等式：每年收到的利息和最后收到的本金，按实际利率r折现回来，等于现在花出去的钱。

用本例来说，就是：59/(1+r) + 59/(1+r)² + 59/(1+r)³ + 59/(1+r)⁴ + (59+1250)/(1+r)⁵ = 1000。

为什么要用插值法？因为这个方程没法直接解，得试几个利率，看哪个让左边等于1000。先试8%，发现左边比1000大；再试12%，发现左边比1000小。那真实利率就在8%和12%之间。再逐步逼近，试到10%的时候刚好对上1000，所以r=10%。插值法本质上就是"猜数字游戏"，只是有系统地猜。

2. 摊余成本的后续计量

每期先算实际利息收入 = 期初摊余成本 × 实际利率。
实际收到的票面利息 = 面值 × 票面利率。
两者的差额就是利息调整的本期摊销额。
期末摊余成本 = 期初摊余成本 + 实际利息收入 - 实际收到的现金。

债权投资有三个明细科目要搞清楚：
- "债权投资——成本"：始终按面值记账，不变。表示到期能收回的本金名义金额。
- "债权投资——利息调整"：记录购买价和面值的差额（折价或溢价），在持有期内逐步摊销到0。
- "债权投资——应计利息"：分期付息时，记录已计提但尚未收到的利息；到期一次还本付息时，累积各期利息。

如果合同现金流量发生变化（比如发行方提前还款），但没有终止确认，企业要重新计算账面余额，差额计入当期损益。重新计算的账面余额，是用修改后的现金流按原实际利率折现的现值。为什么要按原实际利率？因为这是你当初投资时真实的资金成本，不能变。

3. 终止确认或减值时的处理

以摊余成本计量的金融资产，只有在终止确认、重分类、按实际利率法摊销或确认减值时，才会产生计入当期损益的利得或损失。平时持有期间，公允价值的波动不影响账面价值。这也是"摊余成本"这个名字的含义：只管成本摊销，不管市场涨跌。

四、例题四块拆解

例题背景：2×18年1月1日，甲公司花1000万元买了乙公司同日发行的5年期债券12500份，面值总额1250万元，票面年利率4.72%，每年末付息59万元，到期一次性还本。甲公司预计发行方不会提前赎回，将该债券分类为以摊余成本计量的金融资产。

第一块：数据梳理

初始投资成本：1000万元。
债券面值：1250万元。
差额：250万元。这是折价购入，相当于乙公司给了折扣。为什么给折扣？因为票面利率4.72%太低了，市场上类似的债券要给10%的回报投资者才愿意买。所以乙公司只能通过降价（折价发行）来吸引投资者。
票面利率：4.72%，每年收息59万元（1250万×4.72%）。
实际利率r=10%。这意味着，虽然合同上写每年给你59万利息，但你这笔1000万的投资，真实年化回报是10%。

第二块：正常持有情形（情形1）的摊余成本计算

表13-2的核心逻辑：期初摊余成本 × 10% = 实际利息收入。实际利息收入比收到的59万多，差额加到摊余成本里，让摊余成本逐年上升，从1000万涨到1250万。这个过程就是在逐步消化当初折价购入的250万折扣。

表13-2三块解读：

第一块看列：5列分别是年度、期初摊余成本、实际利息收入、现金流入、期末摊余成本。
第二块看关系：实际利息收入 = 期初摊余成本 × 10%。期末摊余成本 = 期初 + 实际利息 - 现金流入。这里的关键是，虽然你只收到59万现金，但账上要确认100万的收益，多出来的41万没拿到手，而是增加了债券的账面价值。
第三块看数字趋势：期初从1000万开始，每年涨一点，最后一年收完本息归零。实际利息收入从100万涨到118万，越来越高，因为本金基数（摊余成本）越来越大。现金流入前4年都是59万，最后一年收本息1309万。

各年数字详解：
2×18年：期初1000万，实际利息=1000×10%=100万，收59万，期末=1000+100-59=1041万。
2×19年：期初1041万，实际利息=1041×10%=104万（约），收59万，期末=1041+104-59=1086万。
2×20年：期初1086万，实际利息=1086×10%=109万（约），收59万，期末=1086+109-59=1136万。
2×21年：期初1136万，实际利息=1136×10%=114万（约），收59万，期末=1136+114-59=1191万。
2×22年：期初1191万。如果按1191×10%=119.1万算，期末=1191+119.1-1309=1.1万，对不上0。所以要做尾数调整：实际利息=1250+59-1191=118万。期末=1191+118-1309=0，刚好归零。

尾数调整的意思：前面几年都四舍五入了，累积有误差。最后一年不能简单地按期初×利率算，而要倒挤一个数，让期末摊余成本刚好等于最终收回金额（或0）。

各年分录解释：

（1）2×18年1月1日购入：
借：债权投资——成本 12500000
贷：银行存款 10000000
贷：债权投资——利息调整 2500000

借方债权投资——成本按面值1250万记，表示将来能收回的本金是1250万。贷方银行存款1000万是实际花的钱。差额250万在贷方利息调整，表示折价购入。这个贷方余额250万要在5年内逐步在借方摊销，最终变成0。

（2）2×18年12月31日计提利息：

应计利息59万是票面利息，按面值和票面利率算。投资收益100万是实际利息收入，按摊余成本和实际利率算。差额41万借记利息调整，就是在摊销当初折价购入的250万折扣。因为折价买了债券，少花了钱，但按实际利率算应该赚100万，只收到59万现金，剩下的41万不是没赚到，而是体现在债券账面价值上涨了——从1000万涨到1041万。

实际收到利息：
借：银行存款 590000
贷：债权投资——应计利息 590000

（3）2×19年12月31日：
借：债权投资——应计利息 590000
借：债权投资——利息调整 450000
贷：投资收益 1040000

借：银行存款 590000
贷：债权投资——应计利息 590000

利息调整摊销额变成45万，比上年的41万多，因为摊余成本从1000万涨到了1041万，基数大了，10%算出来的实际利息更多了。收到的现金还是59万，所以差额更大。

（4）2×20年利息调整摊销50万，2×21年55万，规律一样，逐年递增。

（5）2×22年12月31日：
借：债权投资——应计利息 590000
借：债权投资——利息调整 590000
贷：投资收益 1180000

收利息同上。收本金：

至此，成本1250万清掉，利息调整从250万开始，经过5年摊销（41+45+50+55+59=250万），刚好摊完到0。

第三块：提前收回50%本金情形（情形2）

2×20年1月1日，甲公司预计2×20年末收回50%本金（625万），剩下50%在2×22年末付清。这相当于合同现金流变了，但没终止确认（因为还有一部分继续持有），所以要重新计算账面余额。

重新计算的方法：用修改后的现金流按原实际利率10%折现到2×20年1月1日。
修改后的现金流：
2×20年末收625万本金 + 59万利息（因为2×20年全年本金还是1250万，所以利息还是59万）。
2×21年末收剩余本金625万的利息 = 625万 × 4.72% = 30万（四舍五入）。
2×22年末收剩余本金625万 + 利息30万 = 655万。

现值 = (625+59)/(1+10%) + 30/(1+10%)² + (625+30)/(1+10%)³ = 1139万元。

原来2×20年初的账面余额是1086万元（2×19年末的期末摊余成本）。差额 = 1139 - 1086 = 53万元。这个差额要计入当期损益（投资收益）。

分录：
借：债权投资——利息调整 530000
贷：投资收益 530000

为什么要调？因为合同变了，发行方答应提前还一半，未来的现金流结构变了。按原来的算法，账上只有1086万；但按新的现金流用原利率折现，价值是1139万。多出来的53万是因为提前收回本金让资金更早回笼，按10%的回报率算，早收回的钱更值钱。这53万算是意外之财，计当期收益。

表13-3三块解读：

第一块看列：与表13-2相同，但2×20年的数字变了。
第二块看关系：2×20年期初不是1086万，而是调整后的1139万。实际利息=1139×10%=114万。现金流入=625万本金+59万利息=684万。期末=1139+114-684=569万。
第三块看数字变化：2×21年和2×22年的利息变少了，因为本金只剩一半。2×21年利息=569×10%=57万，收息30万，期末596万。2×22年利息59万（尾数调整625+30-596），收655万，期末0。

第四块：到期一次还本付息情形（情形3）

这次不是每年付息，而是到期一次性给5年利息（每年59万，共295万）加本金1250万，总共1545万。

实际利率计算：(295+1250)/(1+r)⁵ = 1000，解得r≈9.05%。
这个利率比情形1的10%低，因为资金占用时间更长，复利效应没那么强烈。

因为中间不收钱，每年的应计利息都累积在债券里，所以用"债权投资——应计利息"科目累计记载。摊余成本逐年增加，因为没有现金流出，每年的利息都滚入本金继续生息，像滚雪球一样越滚越大。

表13-4三块解读：

第一块看列：同样是5列，但现金流入前4年都是0。
第二块看关系：期初摊余成本按9.05%增长，每年增加的部分就是实际利息收入。因为没有现金流出，期末摊余成本 = 期初 + 实际利息。
第三块看数字：期初1000万，每年滚存，最后一年收1545万归零。利息调整在最后一年一次性摊销718300元。前几年的利息调整摊销额逐年增大（31.5万、39.69万、48.62万、58.36万），反映摊余成本基数越来越大。

【阶段四：自检清单】

1. 是否解释"为什么用实际利率"？是，反映真实资金回报，避免操纵利润。
2. 是否解释实际利率与票面利率区别？是，一个真实回报，一个合同约定。
3. 摊余成本公式是否讲清？是，初始金额加减摊销减已收回本金减减值。
4. 三个明细科目是否解释清楚？是，成本、利息调整、应计利息各自含义和用法。
5. 例题是否四块拆解？是，背景数据→正常持有→提前收回→到期一次付息。
6. 表13-2、表13-3、表13-4是否三块解读？是，列含义→行关系→数字趋势。

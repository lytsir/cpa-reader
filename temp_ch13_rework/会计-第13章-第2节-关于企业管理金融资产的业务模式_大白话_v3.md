锚点ID: 会计-第13章-第2节-关于企业管理金融资产的业务模式

## 这段到底在讲什么？

企业买了金融资产（债券、贷款、承兑汇票等），到底想怎么赚钱？准则把赚钱方式归纳为3种"业务模式"：纯收利息和本金的"收取模式"、既收利息又卖资产赚差价的"双目标模式"、以及纯粹低买高卖的"其他模式"。这段讲的就是怎么判断企业属于哪种模式——要注意组合层面判断、客观证据、合理出售不影响定性等6个要点，并用3个教材例题手把手演示判断过程。业务模式是金融资产分类的"总开关"，直接决定后续用什么会计科目、怎么计量。

---

## 你可能会卡在这里

### 卡点1："业务模式"太抽象，到底看什么？

业务模式不是企业拍脑袋定的，而是一种**客观事实**，看的是企业实际上怎么管理这批资产来产生现金流量。现金流量的来源只有三种可能：

1. **收取合同现金流量**：按合同约定收利息、收本金；
2. **出售金融资产**：把资产卖掉赚差价；
3. **两者兼有**：既收利息，也择机出售。

判断时要看的证据包括：平时怎么考核这批资产？按市场价格考核还是按收了多少钱考核？管理人员的奖金跟什么挂钩？风险怎么管理的？这些都要能拿出真凭实据，不能空口说白话。

### 卡点2：组合能分拆这件事很反直觉——同一批资产为啥能拆？

准则要求在企业**金融资产组合的层次**上确定业务模式，不必逐项确定。但如果一组资产里"鱼龙混杂"——有的想收租、有的想卖——你可以把这组拆成几个小组分别判断。

比如企业买了一个抵押贷款组合，一部分贷款专门收本息，另一部分准备出售套现。这种情况下，可以把该组合拆成两个更小的组合：一个按"收取模式"，一个按"双目标模式"。就像一栋大楼里，有的房间专门出租，有的房间准备装修后卖掉——分开看完全合理。

### 卡点3：收取模式下居然也能卖？什么情况下卖还算"收取模式"？

这是最大的误区。收取模式**不等于**持有至到期！不是说你不卖才叫收取模式，以下几种出售情况仍然可以定性为收取模式：

1. **信用风险恶化时出售**：借款人可能要破产了，你担心收不回钱，赶紧卖掉止损。这种"避险式出售"不改变"收租"的本质。
2. **偶尔出售，哪怕金额大**：比如10年里只卖了一次，单次金额很大，但不代表你平时在倒卖。
3. **频繁出售，但金额都很小**：比如经常小额调整组合，整体价值占比很低。
4. **临近到期出售**：还有3个月到期，提前卖了，卖价跟再等3个月能收到的本息差不多。

核心判断标准：**出售是否反映了业务模式的改变**。如果你能解释清楚为啥卖，而且能证明不代表改变了"收租"的初衷，出售就不影响定性。

### 卡点4：例13-1做了利率互换，还算"老实收租"吗？

例13-1中，甲企业买了一堆已发生信用减值的贷款，同时签了利率互换合同把浮动利率变成固定利率。很多考生觉得：都做了衍生工具了，肯定在投机吧？

**错**。利率互换只是改变了收钱的风险特征（浮动变固定），相当于给收益上了把"保险锁"，并没有改变"收合同现金流量"的本质目的。只要企业还在想方设法催收（打电话、发邮件），初衷仍然是收钱。所以利率互换**不影响**业务模式的判断。

### 卡点5：例13-3的"终止确认"到底是什么意思？

例13-3中，甲公司把承兑汇票拿去背书转让或贴现，分两种情况：

- **不满足终止确认条件**：虽然汇票已经背书给别人，但会计认为你没真正脱手（可能还要承担连带责任），账上还记着这些汇票，同时多了一笔负债。将来你还是靠收票款来实现经济利益，所以是"收取模式"。
- **满足终止确认条件**：会计认为你彻底转手了，票款的风险报酬都转移了，账上已经没有这些汇票了。你的经济利益来源既有收取票款也有出售，所以是"双目标模式"。

记住一句话：**终止确认 = 会计上正式销账，彻底不关我事了**。没终止确认的贴现/背书，只是"形式上转手"，实质上风险还在你身上，所以不算卖。

### 卡点6：三种模式怎么快速区分？

- **收取模式**：卖是偶然的、避险的、临期的，核心是"收"；
- **双目标模式**：卖是日常策略的一部分，既收又卖，两者都不可或缺；
- **其他模式**：纯交易，买来就是为了卖，利息只是顺带收的。

银行管理流动性是最典型的双目标模式：银行要随时准备资产变现应对储户取钱，但又不想资产闲着没收益，所以同时关注合同利息和买卖价差。

## 例题详解

### 例题1：例13-1 贷款组合+利率互换

**条件翻译**：
甲企业购买了一个贷款组合，该组合包含已发生信用减值的贷款。如果贷款不能按时偿付，甲企业将通过邮件、电话或其他方法与借款人联系催收。同时，甲企业签订了一项利率互换合同，将贷款组合的利率由浮动利率转换为固定利率。

**思路拆解**：
考两个判断点：第一，贷款已信用减值，还能算"收取模式"吗？第二，做了利率互换还算"老实收租"吗？

**数字推导**：
本题无具体数字计算，属于纯定性判断题。

关键逻辑链：
- 已发生信用减值 → 收钱难度变大 → 但企业仍在积极催收 → 目的仍是收取合同现金流量 → 业务模式不变
- 签订利率互换 → 浮动利率变固定利率 → 只是改变收钱的风险特征 → 不改变收钱的目的 → 不影响业务模式判断

**陷阱提示**：
1. 不要看到"信用减值"就自动改成其他模式。信用减值只是说明收钱难度增加，不代表企业不想收了。
2. 利率互换是衍生工具，但它在这里的作用是风险管理（锁定利率），不是投机。
3. 企业催收行为（邮件、电话）是"收取合同现金流量"意图的强证据。

**结论**：甲企业管理该贷款组合的业务模式是以收取合同现金流量为目标。利率互换不影响业务模式的判断。

### 例题2：例13-2 银行流动性管理

**条件翻译**：
甲银行持有某金融资产组合以满足其流动性需求。为了降低其管理流动性需求的成本，甲银行高度关注该金融资产组合的回报，包括收取的合同现金流量和出售金融资产的利得或损失。

**思路拆解**：
银行要流动性，说明这些资产随时准备卖；银行又关注合同现金流量，说明收利息也是目的。两者都是实现管理目标不可或缺的，缺了哪个都不行。

**数字推导**：
无具体数字，定性判断。

**陷阱提示**：
1. 别看到"满足流动性需求"就自动脑补成"其他模式"（纯交易模式）。银行是为了流动性而持有，但持有期间也在收利息，"出售"和"收取"都是管理目标的组成部分。
2. 如果是纯交易模式，银行不会关注"收取的合同现金流量"，只会关注买卖价差。

**结论**：甲银行管理该金融资产组合的业务模式是以收取合同现金流量和出售金融资产为目标。

### 例题3：例13-3 承兑汇票背书转让/贴现

**条件翻译**：
甲公司因日常资金管理的需要，在持有承兑汇票的同时，会将部分承兑汇票进行背书转让或贴现。分两个组合管理：
- 组合A：背书转让或贴现时**不满足**金融资产终止确认条件的承兑汇票
- 组合B：背书转让或贴现时**满足**金融资产终止确认条件的承兑汇票

**思路拆解**：
核心看会计上是否"真正卖掉"。组合A没终止确认 = 账上还记着 = 不算卖 = 收取模式；组合B终止确认了 = 账上销掉了 = 算卖 = 双目标模式。

**数字推导**：
无具体数字计算。但补充一个数字场景帮助理解：
- 假设甲公司持有承兑汇票面值1000万元
- 组合A（不满足终止确认）：贴现获得950万元现金，但账上仍确认1000万元应收票据和950万元负债（如短期借款）。经济利益仍来自票据到期收款。
- 组合B（满足终止确认）：贴现获得950万元现金，账上终止确认该票据。经济利益来自两部分：持有期间无（已出售），以及出售对价950万元。

**陷阱提示**：
1. "终止确认"是会计技术判断，不是法律判断。法律上票据已经背书转让了，但会计上可能认为风险没转移。
2. 组合A中，贴现后确认的负债（如短期借款）和原票据同时存在，说明企业实质上仍承担信用风险。
3. 两个组合必须分开管理、分开判断业务模式，不能混为一谈。

**结论**：组合A → 以收取合同现金流量为目标；组合B → 以收取合同现金流量和出售金融资产为目标。

### 三种业务模式对比表

| 业务模式 | 现金流来源 | 出售特征 | 典型场景 |
|---------|-----------|---------|---------|
| 收取合同现金流量 | 仅收取合同现金流量 | 极少，通常是避险或临期 | 持有债券到期收本息 |
| 收取+出售 | 两者兼有 | 出售是日常策略的一部分 | 银行流动性管理 |
| 其他 | 主要通过出售 | 频繁买卖是常态 | 交易性股票、基金 |

### 业务模式判断6个注意点表

| 注意点 | 核心内容 |
|--------|---------|
| 组合层面判断 | 不必逐项确定，可在组合层次判断，必要时可拆分 |
| 多个模式并存 | 一个企业可以同时有多种业务模式 |
| 关键管理人员意图 | 以真正管钱的人的战略定位为准 |
| 客观证据 | 考核方式、风险管理方式、报酬机制等 |
| 不考虑极小概率 | 除非压力情形合理预期会发生，否则不影响判断 |
| 集团和子公司 | 各自按实际情况判断，同一组合通常应一致 |

### 收取模式下"合理出售"的情形表

| 出售情形 | 是否影响"收取模式"定性 | 理由 |
|---------|----------------------|------|
| 信用风险恶化时出售 | 不影响 | 避险式出售，不改变初衷 |
| 偶尔出售（金额大） | 不影响 | 只要不代表业务模式改变 |
| 频繁出售（金额小） | 不影响 | 单独及汇总价值非常小 |
| 临近到期出售（价格接近剩余现金流） | 不影响 | 与持有到期无本质区别 |
| 频繁且大金额出售 | 需重新审视 | 可能反映业务模式已改变 |
